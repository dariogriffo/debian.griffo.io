Was ist Starship?

Starship ist ein schneller, anpassbarer Cross-Shell-Prompt, geschrieben in Rust. Er funktioniert mit Bash, Zsh, Fish, PowerShell und weiteren Shells und zeigt kontextbezogene Informationen — Git-Status, Sprachversionen, Cloud-Kontext und vieles mehr — über einen einzigen, konsistenten und stark konfigurierbaren Prompt, der auch in großen Repositories flott bleibt.

🚀 Warum neueste Versionen wichtig sind: Starship wird aktiv weiterentwickelt, mit neuen Modulen, Integrationen und Performance-Optimierungen. Mit dem neuesten Release erhalten Sie die aktuellsten Prompt-Module und Konfigurationsoptionen.

⚡ Schnell

❓ Häufig gestellte Fragen

Ist Starship in den offiziellen Debian-Paketquellen?

Ja — das offizielle Debian-Archiv liefert Starship 1.22.1. Dieser kostenlose Spiegel liefert das neueste Starship-Release, das mindestens 2 Monate alt ist, für immer kostenlos; das Haupt-Repository auf deb.griffo.io liefert jedes Release wenige Stunden nach Upstream.

Wie installiere ich das neueste Starship unter Debian?

Fügen Sie das deb-free.griffo.io-Repository einmalig hinzu (Anleitung oben) und führen Sie dann aus: sudo apt install starship. Neue Releases kommen über das normale sudo apt upgrade.

Sind die Pakete signiert und wie werden sie gebaut?

Jedes Paket ist mit dem GPG-Schlüssel des Repositories (EA0F721D231FDD3A0A17B9AC7808B4DD62C41256) signiert und wird aus Upstream-Releases in öffentlichen GitHub-Repositories gebaut, die jeder einsehen kann.

Welche Debian-Versionen werden unterstützt?

Debian 12 Bookworm, Debian 13 Trixie, Forky und Sid.
