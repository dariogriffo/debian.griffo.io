¿Qué es Garage?

Garage es un servicio de almacenamiento de objetos distribuido, ligero y compatible con S3. Diseñado para ejecutarse en varias máquinas distribuidas geográficamente, ofrece almacenamiento de objetos resiliente y autoalojado con una API compatible con S3, lo que lo convierte en una sólida alternativa de código abierto a los servicios de almacenamiento de objetos alojados para copias de seguridad, sitios estáticos y datos de aplicaciones.

🚀 Por qué importan las últimas versiones: Garage se desarrolla activamente con mejoras en la replicación, el rendimiento y la cobertura de la API de S3. Ejecutar la última versión te da acceso a las funciones de almacenamiento más recientes y a las correcciones de fiabilidad.

⚡ Características clave de Garage

☁️ API compatible con S3

Compatibilidad total con los clientes, SDKs y herramientas de S3 existentes.

🌐 Distribución geográfica

Replica datos entre múltiples nodos y sitios para mayor resiliencia.

🪶 Ligero

Implementación eficiente en Rust que funciona bien en hardware modesto.

🔒 Autoalojado

Mantén el control total de tus datos en tu propia infraestructura.

🎯 ¿Por qué Garage?

  • Controla tu almacenamiento: Almacenamiento de objetos autoalojado en tu propio hardware
  • Ecosistema S3: Funciona con la amplia variedad de herramientas S3 existentes
  • Resiliente: La replicación multinodo resiste fallos de nodos
  • Eficiente: Bajo consumo de recursos, ideal para clústeres pequeños

🆕 Aspectos destacados

  • Almacenamiento de objetos: Buckets y objetos compatibles con S3
  • Replicación: Redundancia configurable entre nodos
  • Multi-datacenter: Reconocimiento de zonas para la colocación
  • Alojamiento estático: Sirve sitios web estáticos directamente desde buckets

❓ Preguntas frecuentes

¿Está Garage en los repositorios oficiales de Debian?

No — Garage no está empaquetado en los archivos oficiales de Debian. Este espejo gratuito ofrece la versión más reciente de Garage con al menos 2 meses de antigüedad; el repositorio principal en deb.griffo.io ofrece cada versión pocas horas después del upstream.

¿Cómo instalo el último Garage en Debian?

Añade el repositorio deb-free.griffo.io una sola vez con las instrucciones de arriba y ejecuta: sudo apt install garage. Las nuevas versiones llegan con el sudo apt upgrade habitual.

¿Los paquetes están firmados y cómo se compilan?

Cada paquete está firmado con la clave GPG del repositorio (EA0F721D231FDD3A0A17B9AC7808B4DD62C41256) y se compila a partir de versiones upstream en repositorios públicos de GitHub que cualquiera puede inspeccionar.
