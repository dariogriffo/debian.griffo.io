¿Qué es Zed?

Zed es un editor de código de nueva generación diseñado para colaboración de alto rendimiento con humanos e IA. Creado por los desarrolladores de Atom y Tree-sitter, Zed está escrito desde cero en Rust para aprovechar eficientemente múltiples núcleos de CPU y su GPU, ofreciendo una experiencia de edición increíblemente rápida y receptiva.

🚀 Por qué importan las últimas versiones: Zed está evolucionando rápidamente con nuevas características como soporte de depuración nativo, edición agéntica, integración con Git y mejoras de rendimiento en cada lanzamiento. Manténgase actualizado para obtener la mejor experiencia de edición.

🔥 Características Principales de Zed

⚡ Extremadamente Rápido

Escrito desde cero en Rust para aprovechar eficientemente múltiples núcleos de CPU y su GPU. Experimente inicio instantáneo y desplazamiento suave a 120fps.

🤝 Edición Multijugador

Colaboración en tiempo real integrada. Chatee con compañeros de equipo, escriba notas juntos y comparta su pantalla y proyecto sin problemas.

🤖 Integración de IA

Integre LLMs en su flujo de trabajo con edición agéntica, asistente en línea y predicción de edición impulsada por el modelo Zeta.

🐛 Depuración Nativa

Construido sobre el Protocolo de Adaptador de Depuración (DAP) con soporte nativo para depuración en múltiples lenguajes de programación.

🔧 Compatible con Vim

Edición modal de primera clase mediante atajos de Vim, incluyendo características como objetos de texto, marcas y todos sus movimientos favoritos.

🌐 Desarrollo Remoto

Su máquina solo ejecuta la interfaz de Zed, mientras que el código base real se ejecuta en un servidor remoto para un potente desarrollo en la nube.

❓ Preguntas frecuentes

¿Está Zed en los repositorios oficiales de Ubuntu?

No — Zed no está empaquetado en los archivos oficiales de Ubuntu. Este espejo gratuito ofrece la versión más reciente de Zed con al menos 2 meses de antigüedad; el repositorio principal en deb.griffo.io ofrece cada versión pocas horas después del upstream.

¿Cómo instalo el último Zed en Ubuntu?

Añade el repositorio deb-free.griffo.io una sola vez con las instrucciones de arriba y ejecuta: sudo apt install zed. Las nuevas versiones llegan con el sudo apt upgrade habitual.

¿Los paquetes están firmados y cómo se compilan?

Cada paquete está firmado con la clave GPG del repositorio (EA0F721D231FDD3A0A17B9AC7808B4DD62C41256) y se compila a partir de versiones upstream en repositorios públicos de GitHub que cualquiera puede inspeccionar.
