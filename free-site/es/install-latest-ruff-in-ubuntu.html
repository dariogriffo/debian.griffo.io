¿Qué es Ruff?

Ruff es un linter y formateador de código Python extremadamente rápido, escrito en Rust por Astral — el equipo detrás de uv. Con más de 800 reglas integradas y reimplementaciones nativas de plugins populares, un único binario ruff reemplaza a Flake8, isort, Black, pyupgrade, autoflake y decenas de plugins, ejecutándose entre 10 y 100 veces más rápido. Se ha convertido en el linter por defecto del ecosistema Python moderno, adoptado por proyectos como FastAPI, Pandas y Hugging Face.

😤 Dolores de cabeza del tooling de Python: ¿Gestionando flake8 más una docena de plugins, isort peleándose con Black, ejecuciones de lint de varios minutos en CI, e instalando linters en cada virtualenv? Ruff es un único binario sin dependencias que lo hace todo en milisegundos — e instalarlo con apt lo deja disponible en todo el sistema, para cada proyecto y venv, sin pip.

⚡ Características principales de Ruff

🏎️ De 10 a 100 veces más rápido

Analiza bases de código grandes en milisegundos donde las herramientas tradicionales tardan minutos — lo bastante rápido para ejecutarse en cada guardado y en cada pipeline de CI.

🧰 Una herramienta, más de 800 reglas

Reimplementa Flake8, isort, pyupgrade, pydocstyle, eradicate y más — una sola configuración en pyproject.toml reemplaza una pila de plugins.

🖤 Formateador drop-in

ruff format es un formateador compatible con Black — el mismo estilo, órdenes de magnitud más rápido, sin herramientas adicionales que instalar.

🔧 Correcciones automáticas

Cientos de reglas incluyen correcciones automáticas: imports sin usar, sintaxis actualizada, imports ordenados — ruff check --fix y listo.

🐍 Consciente de la versión

Apunta a cualquier Python desde 3.7 en adelante con comportamiento de reglas por versión, configuración jerárquica apta para monorepos y anulaciones por archivo.

🔌 Integraciones con editores

Un language server integrado (ruff server) impulsa integraciones de primera clase con VS Code, Zed, Neovim y Helix desde el mismo binario.

🏆 ¿Por qué instalarlo con apt?

  • A nivel de sistema, sin pip: un binario para cada proyecto, venv, trabajo de CI y usuario — sin instalaciones por proyecto que mantener sincronizadas
  • Perfecto para imágenes Docker/CI: incorpora el repositorio en tus imágenes base y cada pipeline obtiene un ruff actualizado automáticamente
  • Sin toolchain: sin Rust, sin cargo, sin pip — un repositorio apt firmado con binarios precompilados para 8 arquitecturas
  • Actualizaciones automáticas: ruff publica versiones con frecuencia; apt upgrade te mantiene al día para que tus reglas nunca queden obsoletas
  • Combina con uv: el gestor de paquetes de Astral también se sirve aquí — el toolchain completo del Python moderno desde un solo repositorio

❓ Preguntas frecuentes

¿Está Ruff en los repositorios oficiales de Ubuntu?

No — Ruff no está empaquetado en los archivos oficiales de Ubuntu. Este espejo gratuito ofrece la versión más reciente de Ruff con al menos 2 meses de antigüedad; el repositorio principal en deb.griffo.io ofrece cada versión pocas horas después del upstream.

¿Cómo instalo el último Ruff en Ubuntu?

Añade el repositorio deb-free.griffo.io una sola vez con las instrucciones de arriba y ejecuta: sudo apt install ruff. Las nuevas versiones llegan con el sudo apt upgrade habitual.

¿Los paquetes están firmados y cómo se compilan?

Cada paquete está firmado con la clave GPG del repositorio (EA0F721D231FDD3A0A17B9AC7808B4DD62C41256) y se compila a partir de versiones upstream en repositorios públicos de GitHub que cualquiera puede inspeccionar.

¿Qué versiones de Ubuntu son compatibles?

Ubuntu 22.04 Jammy, 24.04 Noble, 25.10 Questing y 26.04 Resolute.
