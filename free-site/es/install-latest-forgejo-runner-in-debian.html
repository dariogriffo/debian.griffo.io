¿Qué es Forgejo Runner?

Forgejo Runner es el runner de CI para Forgejo Actions. Es un daemon ligero que se conecta a tu instancia de Forgejo, recoge los trabajos de workflow en cola y los ejecuta en tu propia infraestructura — usando Docker, LXC o directamente el host — dándote un pipeline de CI/CD totalmente autoalojado y compatible con GitHub Actions.

🚀 Por qué importan las últimas versiones: El runner sigue de cerca los lanzamientos de Forgejo, añadiendo soporte para nuevas funciones de workflow, mejor aislamiento y correcciones de errores. Mantenerlo actualizado garantiza compatibilidad con las últimas capacidades de Forgejo Actions.

⚡ Características clave de Forgejo Runner

🔁 Compatible con Actions

Ejecuta workflows usando la misma sintaxis y ecosistema de acciones que GitHub Actions.

🐳 Ejecución flexible

Ejecuta trabajos en contenedores Docker, LXC o directamente en la máquina host.

🔐 Autoalojado

Mantén todo tu CI/CD en tu propia infraestructura, sin dependencia de terceros.

⚖️ Escalable

Registra varios runners con etiquetas para distribuir y dirigir las cargas de trabajo.

🎯 ¿Por qué Forgejo Runner?

  • Controla tu CI: Ejecuta pipelines en hardware que tú controlas
  • Ecosistema de Actions: Reutiliza definiciones de workflow existentes al estilo GitHub
  • Aislamiento flexible: Ejecución en Docker, LXC o host según tus necesidades
  • Ligero: Un único binario de Go, sencillo de desplegar

🆕 Aspectos destacados

  • Ejecución de trabajos: Ejecuta los workflows en cola de Forgejo Actions
  • Etiquetas: Dirige trabajos a runners específicos
  • Soporte de contenedores: Compilaciones aisladas mediante Docker
  • Modo daemon: Se ejecuta continuamente como servicio

❓ Preguntas frecuentes

¿Está Forgejo Runner en los repositorios oficiales de Debian?

No — Forgejo Runner no está empaquetado en los archivos oficiales de Debian. Este espejo gratuito ofrece la versión más reciente de Forgejo Runner con al menos 2 meses de antigüedad; el repositorio principal en deb.griffo.io ofrece cada versión pocas horas después del upstream.

¿Cómo instalo el último Forgejo Runner en Debian?

Añade el repositorio deb-free.griffo.io una sola vez con las instrucciones de arriba y ejecuta: sudo apt install forgejo-runner. Las nuevas versiones llegan con el sudo apt upgrade habitual.

¿Los paquetes están firmados y cómo se compilan?

Cada paquete está firmado con la clave GPG del repositorio (EA0F721D231FDD3A0A17B9AC7808B4DD62C41256) y se compila a partir de versiones upstream en repositorios públicos de GitHub que cualquiera puede inspeccionar.

¿Qué versiones de Debian son compatibles?

Debian 12 Bookworm, Debian 13 Trixie, Forky y Sid.
