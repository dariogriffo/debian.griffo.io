¿Qué es Fastfetch?

Fastfetch es una herramienta estilo neofetch para obtener información del sistema y mostrarla de forma hermosa en tu terminal — SO, kernel, hardware, tiempo de actividad, paquetes, entorno de escritorio y mucho más, junto al logo de tu distro. Escrita principalmente en C con el rendimiento y la personalización en mente, se convirtió en la sucesora de facto cuando neofetch fue archivado, y se desarrolla activamente con lanzamientos frecuentes.

😤 El problema de la versión del archivo: Debian sí incluye fastfetch — congelado en la versión que existía cuando se publicó la release. Upstream avanza rápido: nuevos logos, nuevos módulos de detección y correcciones para hardware nuevo llegan constantemente, así que la versión del archivo se queda atrás en silencio. Este repositorio ofrece builds actualizados de upstream manteniendo exactamente la misma disposición de archivos que el paquete oficial de Debian, por lo que el cambio es transparente en ambas direcciones.
🆓 Gratis para siempre: Debian y Ubuntu congelan las versiones de los paquetes al publicar cada versión — este espejo gratuito ofrece la versión más reciente con al menos 2 meses de antigüedad, sin cuenta ni autenticación, y publica las correcciones de seguridad de inmediato. ¿Necesitas cada versión pocas horas después del upstream? ⚡ Características principales de Fastfetch

🏎️ Rápido por diseño

Escrito en C con detección concurrente — muestra la información completa del sistema en milisegundos, una fracción del tiempo de arranque de neofetch.

🧩 Más de 50 módulos de información

SO, kernel, CPU, GPU, memoria, discos, batería, pantalla, DE/WM, terminal, paquetes, reproductor multimedia y muchos más — cada uno configurable individualmente.

🎨 Presets incluidos

Este paquete incluye todos los presets integrados en /usr/share/fastfetch/presets/ — incluido un preset neofetch fiel para una familiaridad instantánea.

📝 Configuración en JSONC

Un único archivo de configuración JSONC documentado controla módulos, formato y logos. Genera un punto de partida con fastfetch --gen-config.

🖼️ Logos flexibles

Cientos de logos de distros integrados, además de logos en imagen mediante los protocolos sixel/kitty/iTerm, arte ASCII personalizado, o ninguno en absoluto.

🐚 Completados de shell

Completados para bash, zsh y fish más una página de manual — instalados en las rutas estándar de Debian, igual que el paquete del archivo.

🏆 ¿Por qué este repositorio en lugar de la versión del archivo?

  • Versión actual: la versión más reciente con al menos 2 meses de antigüedad, gratis para siempre, sin quedar congelada durante toda la vida de una release de Debian
  • Misma disposición: alineado archivo por archivo con el paquete oficial de Debian — presets, completados y página de manual en los lugares estándar
  • Cambio directo: instalar desde este repositorio simplemente actualiza el paquete del archivo; al quitar el repositorio se vuelve atrás limpiamente
  • Actualizaciones automáticas: las nuevas versiones llegan con tu apt upgrade habitual
  • Multi-arch: amd64, arm64, armel, armhf, ppc64el, s390x, riscv64 e i386

¿Está Fastfetch en los repositorios oficiales de Debian?

Sí — el archivo oficial de Debian trae Fastfetch 2.62.1. Este espejo gratuito ofrece la versión más reciente de Fastfetch con al menos 2 meses de antigüedad, gratis para siempre; el repositorio principal en deb.griffo.io ofrece cada versión pocas horas después del upstream.

¿Cómo instalo el último Fastfetch en Debian?

Añade el repositorio deb-free.griffo.io una sola vez con las instrucciones de arriba y ejecuta: sudo apt install fastfetch. Las nuevas versiones llegan con el sudo apt upgrade habitual.

¿Los paquetes están firmados y cómo se compilan?
