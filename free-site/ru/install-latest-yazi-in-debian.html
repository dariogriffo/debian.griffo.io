Что такое Yazi?

Yazi (означает «утка» 🦆) — это молниеносный файловый менеджер для терминала, написанный на Rust, основанный на неблокирующем асинхронном I/O. Он стремится обеспечить эффективный, удобный и настраиваемый опыт управления файлами, превосходящий традиционные файловые менеджеры благодаря современной архитектуре и инновационным функциям.

🚀 Почему важны последние версии: Yazi активно развивается с частыми релизами, содержащими оптимизации производительности, новые плагины, улучшенную поддержку предпросмотра изображений и улучшения рабочего процесса. Последние версии включают значительные архитектурные улучшения и новые функции, существенно повышающие удобство работы.

⚡ Ключевые возможности Yazi

🚀 Полная асинхронность

Все операции I/O асинхронны, задачи CPU распределены по нескольким потокам, что максимально использует доступные ресурсы для молниеносной производительности.

🖼️ Предпросмотр изображений

Встроенная поддержка нескольких протоколов изображений, включая Kitty, iTerm2, Sixel и другие. Предпросмотр изображений, видео, PDF-файлов и кода прямо в терминале.

🔌 Система плагинов

Параллельная система плагинов на основе Lua с плагинами UI, функциональными плагинами, настраиваемыми средствами предпросмотра и встроенным менеджером пакетов.

📡 Распределение данных

Клиент-серверная архитектура с моделью публикации-подписки на основе Lua для межэкземплярной коммуникации и сохранения состояния.

🧰 Интеграция с инструментами

Бесшовная интеграция с ripgrep, fd, fzf, zoxide и другими современными инструментами командной строки для улучшенных рабочих процессов.

🎨 Настройка

Система тем, настраиваемые макеты, Vim-подобные сочетания клавиш, поддержка мыши и обширные параметры конфигурации.

🏆 Почему Yazi такой быстрый

  • Асинхронный I/O: Неблокирующие операции предотвращают зависание UI
  • Многопоточность: Задачи CPU распределены по потокам
  • Умное кэширование: Механизмы предзагрузки и кэширования
  • Производительность Rust: Язык системного программирования с безопасной работой с памятью
  • Оптимизированный рендеринг: Эффективные алгоритмы рендеринга терминала
🚀 Как связаны два зеркала: основной репозиторий deb.griffo.io публикует новые версии upstream в течение нескольких часов; это бесплатное зеркало отдаёт каждую версию, когда ей исполняется не менее 2 месяцев — исправления безопасности немедленно. Обычный apt upgrade держит вас в актуальном состоянии в пределах этого окна.

# Start Yazi in current directory yazi # Start in specific directory yazi /path/to/directory # File chooser mode (returns selected file) yazi --chooser-file=/tmp/chosen

Основные сочетания клавиш:

# Навигация j/k или ↑/↓ - Вверх/вниз h/l или ←/→ - Войти/выйти из директории g/G - В начало/конец / - Поиск файлов # Операции с файлами y - Копировать (yank) x - Вырезать p - Вставить d - Удалить r - Переименовать c - Создать файл/директорию # Виды t - Новая вкладка 1-9 - Переключить на вкладку z - Показать/скрыть скрытые файлы

Расширенные функции:

# Массовые операции space - Выбрать/снять выбор файла v - Визуальный режим выбора ctrl+a - Выбрать всё # Управление плагинами : - Режим команд :plugin - Операции с плагинами # Интеграция :zoxide - Прыжок с zoxide :fzf - Нечёткий поиск файлов

🚀 Почему стоит выбрать deb.griffo.io?

📊 Сравнение репозиториев:
  • Официальный Debian: Часто устаревшие версии без новейших функций
  • Cargo Install: Требует цепочки инструментов Rust и времени на компиляцию
  • deb.griffo.io (main): Последняя версия доступна немедленно
  • ✅ Последние функции: Получайте новейшие функции Yazi и улучшения производительности
  • ✅ Без компиляции Rust: Готовые бинарные файлы, цепочка инструментов не нужна
  • ✅ Автоматические обновления: Пакеты обновляются автоматически; в бесплатной линии — не позже 2 месяцев после релиза, исправления безопасности немедленно
  • ✅ Правильные зависимости: Корректно упакован со всеми необходимыми библиотеками
  • ✅ Мульти-дистрибутив: Работает на Bookworm, Trixie и Sid
  • ✅ Готов к плагинам: Включает все зависимости для системы плагинов

❓ Часто задаваемые вопросы

Есть ли Yazi в официальных репозиториях Debian?

Нет — Yazi не упакован в официальных архивах Debian. Это бесплатное зеркало предлагает новейший релиз Yazi возрастом не менее 2 месяцев; основной репозиторий deb.griffo.io предлагает каждый релиз через несколько часов после upstream.

Как установить последнюю версию Yazi в Debian?

Добавьте репозиторий deb-free.griffo.io один раз по инструкции выше, затем выполните: sudo apt install yazi. Новые версии приходят через обычный sudo apt upgrade.

Подписаны ли пакеты и как они собираются?

Каждый пакет подписан GPG-ключом репозитория (EA0F721D231FDD3A0A17B9AC7808B4DD62C41256) и собирается из релизов upstream в публичных репозиториях GitHub, которые может проверить любой.

Какие версии Debian поддерживаются?

Debian 12 Bookworm, Debian 13 Trixie, Forky и Sid.
