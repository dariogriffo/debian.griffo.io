Что такое Garage?

Garage — это лёгкое, S3-совместимое распределённое объектное хранилище. Разработанное для работы на нескольких географически распределённых машинах, оно обеспечивает отказоустойчивое, самостоятельно размещаемое объектное хранилище с S3-совместимым API, что делает его надёжной альтернативой с открытым исходным кодом облачным хранилищам объектов для резервного копирования, статических сайтов и данных приложений.

🚀 Почему важны последние версии: Garage активно развивается — с улучшениями репликации, производительности и покрытия S3 API. Использование последней версии даёт вам новейшие функции хранения и исправления надёжности.

⚡ Ключевые возможности Garage

☁️ S3-совместимый API

Полная совместимость с существующими S3-клиентами, SDK и инструментами.

🌐 Гео-распределённость

Репликация данных между несколькими узлами и площадками для отказоустойчивости.

🪶 Лёгкость

Эффективная реализация на Rust, хорошо работающая на скромном оборудовании.

🔒 Самостоятельный хостинг

Полный контроль над своими данными на собственной инфраструктуре.

🎯 Почему Garage?

  • Владейте своим хранилищем: Самостоятельно размещаемое объектное хранилище на собственном оборудовании
  • Экосистема S3: Работает с широким набором существующих инструментов S3
  • Отказоустойчивость: Репликация между несколькими узлами переживает отказы узлов
  • Эффективность: Низкое потребление ресурсов, идеально для небольших кластеров

🆕 Основные возможности

  • Объектное хранилище: S3-совместимые бакеты и объекты
  • Репликация: Настраиваемая избыточность между узлами
  • Мульти-датацентр: Учёт зон при размещении данных
  • Статический хостинг: Обслуживание статических сайтов прямо из бакетов

❓ Часто задаваемые вопросы

Есть ли Garage в официальных репозиториях Ubuntu?

Нет — Garage не упакован в официальных архивах Ubuntu. Это бесплатное зеркало предлагает новейший релиз Garage возрастом не менее 2 месяцев; основной репозиторий deb.griffo.io предлагает каждый релиз через несколько часов после upstream.

Как установить последнюю версию Garage в Ubuntu?

Добавьте репозиторий deb.griffo.io один раз по инструкции выше, затем выполните: sudo apt install garage. Новые версии приходят через обычный sudo apt upgrade.

Подписаны ли пакеты и как они собираются?

Каждый пакет подписан GPG-ключом репозитория (EA0F721D231FDD3A0A17B9AC7808B4DD62C41256) и собирается из релизов upstream в публичных репозиториях GitHub, которые может проверить любой.

Какие версии Ubuntu поддерживаются?

Ubuntu 22.04 Jammy, 24.04 Noble, 25.10 Questing и 26.04 Resolute.
