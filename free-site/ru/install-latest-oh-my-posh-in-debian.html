Что такое Oh My Posh?

Oh My Posh — это движок тем приглашения командной строки для любой оболочки, созданный Яном Де Доббелером и написанный на Go. Один движок, один формат тем — одно и то же красивое, информативное приглашение в bash, zsh, fish, PowerShell, Nushell, elvish, xonsh и cmd. Набрав десятки тысяч звёзд на GitHub и обладая огромной экосистемой тем, он стал стандартным способом получить единообразное приглашение на каждой машине и в каждой оболочке, которыми вы пользуетесь.

😤 Проблемы с приглашением командной строки: Вручную пишете escape-коды для PS1? Привязаны к фреймворку приглашений, работающему только в одной оболочке? Пересобираете приглашение с нуля каждый раз, когда переключаетесь между zsh дома и bash на серверах? Oh My Posh даёт одну конфигурацию, которая следует за вами повсюду.
🚀 Всегда впереди официальных архивов: Debian и Ubuntu замораживают версии пакетов при выпуске релиза — этот репозиторий публикует новые версии обычно в течение нескольких часов после релиза upstream. Простой apt upgrade всегда держит вас на последней версии.

⚡ Ключевые возможности Oh My Posh

🐚 Любая оболочка, одна конфигурация

Идентичное приглашение в bash, zsh, fish, PowerShell, Nushell, elvish, xonsh и cmd. Настройте один раз, используйте везде — в том числе по SSH.

🎨 125+ тем в комплекте

Этот пакет устанавливает все встроенные темы в /usr/share/oh-my-posh/themes/ — готовы к просмотру и использованию офлайн, без дополнительных загрузок.

🧩 Богатые сегменты

Статус Git, коды выхода, время выполнения, батарея, контекст kubernetes, облачные аккаунты и версии языков (Go, Python, Node, Rust и десятки других) — показываются только тогда, когда это уместно.

📝 Ваша тема, ваш формат

Пишите собственные темы в JSON, YAML или TOML с документированной схемой. Начните с любой встроенной темы и сделайте её своей.

✨ Transient-приглашения и подсказки

Transient-приглашения сохраняют историю прокрутки чистой, подсказки реагируют на то, что вы вводите, а сегменты с выравниванием по правому краю используют всю ширину терминала.

⚡ Быстрый по своей природе

Один бинарный файл на Go с параллельным рендерингом сегментов — приглашение остаётся мгновенным даже со множеством включённых сегментов.

🏆 Зачем устанавливать через apt?

  • Без curl-pipe-bash: репозиторий с GPG-подписью вместо перенаправления скриптов установки в вашу оболочку
  • Темы в комплекте: все встроенные темы попадают в /usr/share/oh-my-posh/themes/ и соответствуют версии бинарного файла
  • Автоматические обновления: новые релизы upstream приходят с обычным apt upgrade
  • Чистое удаление: apt remove убирает всё за собой — никаких потерянных бинарных файлов в ~/bin
  • Мульти-архитектура: amd64, arm64 и armhf — одна и та же настройка на десктопе и на Raspberry Pi

🚀 Настройте свою оболочку

Выберите тему из /usr/share/oh-my-posh/themes/ и инициализируйте оболочку. Для bash добавьте это в ~/.bashrc:

eval "$(oh-my-posh init bash --config /usr/share/oh-my-posh/themes/jandedobbeleer.omp.json)"

Для zsh добавьте ту же строку с zsh в ~/.zshrc; для fish добавьте oh-my-posh init fish --config /usr/share/oh-my-posh/themes/jandedobbeleer.omp.json | source в ~/.config/fish/config.fish. Большинство тем лучше всего выглядят с Nerd Font — или выполните oh-my-posh font install.

❓ Часто задаваемые вопросы

Есть ли Oh My Posh в официальных репозиториях Debian?

Нет — Oh My Posh не упакован в официальных архивах Debian. Это бесплатное зеркало предлагает новейший релиз Oh My Posh возрастом не менее 2 месяцев; основной репозиторий deb.griffo.io предлагает каждый релиз через несколько часов после upstream.

Как установить последнюю версию Oh My Posh в Debian?

Добавьте репозиторий deb.griffo.io один раз по инструкции выше, затем выполните: sudo apt install oh-my-posh. Новые версии приходят через обычный sudo apt upgrade.

Подписаны ли пакеты и как они собираются?

Каждый пакет подписан GPG-ключом репозитория (EA0F721D231FDD3A0A17B9AC7808B4DD62C41256) и собирается из релизов upstream в публичных репозиториях GitHub, которые может проверить любой.

Какие версии Debian поддерживаются?

Debian 12 Bookworm, Debian 13 Trixie, Forky и Sid.
