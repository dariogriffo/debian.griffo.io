Что такое Fastfetch?

Fastfetch — это инструмент в стиле neofetch для сбора информации о системе и её красивого отображения в терминале — ОС, ядро, оборудование, время работы, пакеты, окружение рабочего стола и многое другое, рядом с логотипом вашего дистрибутива. Написанный преимущественно на C с упором на производительность и настраиваемость, он стал де-факто преемником neofetch после его архивации и активно развивается с частыми релизами.

😤 Проблема версии из архива: Старые релизы Ubuntu вообще не содержат fastfetch, а там, где он есть в архиве, он заморожен на момент импорта. Upstream развивается быстро: новые логотипы, новые модули определения и исправления для нового оборудования появляются постоянно, поэтому версия из архива незаметно отстаёт. Это бесплатное зеркало предоставляет сборки upstream не старше 2 месяцев, сохраняя точно такую же раскладку файлов, как в официальном пакете архива, так что переключение бесшовно в обе стороны.
🚀 Всегда впереди официальных архивов: Debian и Ubuntu замораживают версии пакетов при выпуске релиза — этот репозиторий публикует новые версии обычно в течение нескольких часов после релиза upstream. Простой apt upgrade всегда держит вас на последней версии.

⚡ Ключевые возможности Fastfetch

🏎️ Быстрый по замыслу

Написан на C с параллельным определением — выводит полную информацию о системе за миллисекунды, за долю времени запуска neofetch.

🧩 50+ информационных модулей

ОС, ядро, CPU, GPU, память, диски, батарея, дисплей, DE/WM, терминал, пакеты, медиаплеер и многие другие — каждый настраивается отдельно.

🎨 Пресеты в комплекте

Этот пакет поставляет все встроенные пресеты в /usr/share/fastfetch/presets/ — включая точный пресет neofetch для мгновенной привычности.

📝 Конфигурация JSONC

Один документированный конфигурационный файл JSONC управляет модулями, форматированием и логотипами. Создайте отправную точку командой fastfetch --gen-config.

🖼️ Гибкость логотипов

Сотни встроенных логотипов дистрибутивов, плюс логотипы-изображения через протоколы sixel/kitty/iTerm, собственный ASCII-арт или вовсе без логотипа.

🐚 Автодополнение для оболочек

Автодополнение для bash, zsh и fish плюс man-страница — устанавливаются в стандартные пути Debian, как и в пакете из архива.

🏆 Почему этот репозиторий, а не версия из архива?

  • Актуальная версия: новые релизы upstream в течение нескольких часов, без заморозки на весь срок жизни релиза Debian
  • Та же раскладка: файл в файл совпадает с официальным пакетом Debian — пресеты, автодополнение и man-страница на стандартных местах
  • Бесшовное переключение: установка из этого репозитория просто обновляет пакет из архива; после удаления репозитория происходит чистый откат
  • Автоматические обновления: новые релизы приходят с обычным apt upgrade
  • Мульти-архитектура: amd64, arm64, armhf, ppc64el, s390x и riscv64

❓ Часто задаваемые вопросы

Есть ли Fastfetch в официальных репозиториях Ubuntu?

Да — официальный архив Ubuntu поставляет Fastfetch 2.57.1. Это бесплатное зеркало предлагает новейший релиз Fastfetch возрастом не менее 2 месяцев, бесплатно навсегда; основной репозиторий deb.griffo.io предлагает каждый релиз через несколько часов после upstream.

Как установить последнюю версию Fastfetch в Ubuntu?

Добавьте репозиторий deb.griffo.io один раз по инструкции выше, затем выполните: sudo apt install fastfetch. Новые версии приходят через обычный sudo apt upgrade.

Подписаны ли пакеты и как они собираются?

Каждый пакет подписан GPG-ключом репозитория (EA0F721D231FDD3A0A17B9AC7808B4DD62C41256) и собирается из релизов upstream в публичных репозиториях GitHub, которые может проверить любой.

Какие версии Ubuntu поддерживаются?

Ubuntu 22.04 Jammy, 24.04 Noble, 25.10 Questing и 26.04 Resolute.
