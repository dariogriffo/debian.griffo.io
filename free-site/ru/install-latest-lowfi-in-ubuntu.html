Что такое lowfi?

lowfi — это маленькое Rust-приложение с единственной целью: воспроизводить lofi. Созданное talwat, оно делает это максимально просто — никаких альбомов, рекламы или отвлечений, только чистая lofi-музыка для программирования и концентрации. Приложение разработано с философией «suckless» — предоставляет ровно то, что нужно, и ничего лишнего.

🎯 Философия дизайна: «Мне очень не нравятся современные музыкальные платформы, и я хотел небольшое, «suckless»-приложение, которое просто воспроизводило бы случайную lofi-музыку без видео.» — talwat, создатель lowfi
🚀 Почему важны последние версии: lowfi активно развивается с частыми выпусками, содержащими новые источники треков, улучшения производительности и расширенные функции. Последние версии включают поддержку пользовательских списков треков, интеграцию MPRIS и улучшенную сетевую устойчивость для непрерывных сессий программирования.

⚡ Ключевые возможности lowfi

🎵 Чистая простота

Приложение с единственной целью — просто воспроизводить lofi-музыку. Никаких сложных интерфейсов и отвлечений, идеально для поддержания потока программирования.

🌐 Устойчивость к сети

Буферизует 5 целых треков единовременно вместо потоковой передачи фрагментами. Разработан для нестабильных сетей и непрерывного прослушивания.

🎛️ Поддержка медиаклавиш

Работает с медиаклавишами и MPRIS (playerctl). Управляйте воспроизведением, не покидая редактор и не нарушая концентрацию.

📻 Пользовательские списки треков

Поддержка пользовательских списков треков, включая Chillhop, локальные файлы и плейлисты от сообщества для разнообразия.

⚡ Лёгкость

Крошечный бинарный файл с минимальным потреблением ресурсов. Не замедлит среду разработки и не нагрузит систему.

🎨 Настраиваемый интерфейс

Минималистичный режим, безрамочный дисплей, поддержка альтернативного экрана и настраиваемая ширина плеера для разных терминалов.

🎯 Примеры базового использования

Начать прослушивание:

# Just start lowfi - that's it! lowfi # Start in minimalist mode (no bottom bar) lowfi --minimalist # Start paused lowfi --paused # Use alternate terminal screen lowfi --alternate

Пользовательские списки треков:

# Use Chillhop instead of Lofi Girl lowfi --track-list chillhop # Use custom track list file lowfi --track-list ~/my-lofi-tracks.txt # Use community track lists lowfi --track-list minipop

Расширенные параметры:

# Borderless UI lowfi --borderless # Custom player width lowfi --width 5 # Debug mode (for troubleshooting) lowfi --debug # Custom buffer size lowfi --buffer-size 10

🚀 Почему выбрать deb.griffo.io?

📊 Сравнение способов установки:
  • Официальный Ubuntu: Недоступен в официальных репозиториях
  • Cargo Install: Требует цепочки инструментов Rust и компиляции
  • Ручной бинарный файл: Ручные обновления без управления пакетами
  • deb.griffo.io (main): Последняя версия с автоматическими обновлениями
  • ✅ Последние функции: Новейшие источники треков и улучшения плеера
  • ✅ Без компиляции Rust: Готовые бинарные файлы, цепочка инструментов не нужна
  • ✅ Автоматические обновления: Пакеты обновляются автоматически; в бесплатной линии — не позже 2 месяцев после релиза, исправления безопасности немедленно
  • ✅ Правильная аудионастройка: Корректно упакован с ALSA и аудиозависимостями
  • ✅ Мультидистрибутив: Работает на Jammy и Noble
  • ✅ Простое обслуживание: Стандартные команды apt для обновлений

❓ Часто задаваемые вопросы

Есть ли lowfi в официальных репозиториях Ubuntu?

Нет — lowfi не упакован в официальных архивах Ubuntu. Это бесплатное зеркало предлагает новейший релиз lowfi возрастом не менее 2 месяцев; основной репозиторий deb.griffo.io предлагает каждый релиз через несколько часов после upstream.

Как установить последнюю версию lowfi в Ubuntu?

Добавьте репозиторий deb-free.griffo.io один раз по инструкции выше, затем выполните: sudo apt install lowfi. Новые версии приходят через обычный sudo apt upgrade.

Подписаны ли пакеты и как они собираются?

Каждый пакет подписан GPG-ключом репозитория (EA0F721D231FDD3A0A17B9AC7808B4DD62C41256) и собирается из релизов upstream в публичных репозиториях GitHub, которые может проверить любой.

Какие версии Ubuntu поддерживаются?

Ubuntu 22.04 Jammy, 24.04 Noble, 25.10 Questing и 26.04 Resolute.
