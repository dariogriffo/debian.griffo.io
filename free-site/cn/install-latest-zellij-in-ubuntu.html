什么是 Zellij？

Zellij 是一款用 Rust 编写的终端工作区与复用器。作为 tmux 和 screen 的现代替代品，它让您可以将终端拆分为多个面板和标签页，保存并恢复布局，并通过 WebAssembly 插件系统进行扩展——所有这一切都配有易于发现的快捷键提示和友好的用户界面，让上手变得轻松。

🚀 为什么最新版本很重要：Zellij 正在积极开发中，不断推出新的布局功能、插件能力和性能改进。运行最新版本可以让您第一时间获得最新的工作区功能和修复。

⚡ Zellij 的核心功能

🪟 面板与标签页

将终端拆分为多个面板，并通过多个标签页组织您的工作。

📐 布局

为您的项目定义并恢复可复现的工作区布局。

🔌 插件系统

通过 WebAssembly 插件扩展 Zellij，实现自定义功能。

🧭 易于发现的界面

屏幕提示让快捷键易学易记。

🎯 为什么选择 Zellij？

  • 新手友好：屏幕上的快捷键提示降低了学习曲线
  • 强大的布局：通过配置文件定义可复现的工作区
  • 可扩展：WebAssembly 插件系统
  • 现代化：基于 Rust 的快速实现，默认配置合理

🆕 亮点

  • 复用功能：在单个终端中使用多个面板和标签页
  • 会话管理：随时从运行中的会话分离并重新连接
  • 布局：保存并加载工作区排列方式
  • 插件：基于 WASM 的可扩展性

❓ 常见问题

Zellij 在官方 Ubuntu 软件源中吗？

不在 — 官方 Ubuntu 源没有打包 Zellij。本免费镜像提供发布至少 2 个月的最新 Zellij 版本；主仓库 deb.griffo.io 在上游发布后数小时内提供每个版本。

如何在 Ubuntu 上安装最新版 Zellij？

按上方说明一次性添加 deb-free.griffo.io 仓库，然后运行：sudo apt install zellij。之后通过常规的 sudo apt upgrade 获取新版本。

软件包是否有签名？如何构建？

每个软件包都使用仓库的 GPG 密钥（EA0F721D231FDD3A0A17B9AC7808B4DD62C41256）签名，并基于上游发布在公开的 GitHub 仓库中构建，任何人都可以审查。

支持哪些 Ubuntu 版本？

Ubuntu 22.04 Jammy、24.04 Noble、25.10 Questing 和 26.04 Resolute。
