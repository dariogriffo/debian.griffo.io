什么是 Atuin？

Atuin 会用可搜索的 SQLite 数据库替代您的 shell 历史记录，提供强大的全文搜索、可选的跨设备端到端加密同步，以及实用的统计信息。它会为每条命令记录额外的上下文——退出码、耗时、目录等——并支持 Bash、Zsh、Fish 等多种 shell。

🚀 为什么最新版本很重要：Atuin 正在积极开发中，持续改进搜索、同步和交互式界面。运行最新版本可以让您获得最新的历史记录功能和修复。

⚡ Atuin 的核心功能

🔍 强大的搜索

对整个 shell 历史记录进行全文、模糊和前缀搜索。

🔐 加密同步

可选的端到端加密同步，在您所有设备间同步历史记录。

📊 丰富的上下文

为每条命令记录退出码、耗时、目录等信息。

🐚 跨 Shell 支持

支持 Bash、Zsh、Fish 等主流 shell。

🎯 为什么选择 Atuin？

  • 更好的历史记录：可搜索的 SQLite 历史记录，永不静默截断
  • 跨设备同步：端到端加密，可自托管的同步服务
  • 上下文丰富：了解每条命令的执行时间、地点和方式
  • 统计信息：深入了解您最常用的命令

🆕 亮点功能

  • SQLite 历史记录：持久且可查询的命令历史
  • 交互式搜索：绑定到向上箭头键的模糊搜索
  • 同步：可选的加密多设备同步
  • 统计：命令使用统计信息
