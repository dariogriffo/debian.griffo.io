什么是 Helix？

Helix 是一款使用 Rust 编写的后现代模态文本编辑器。它受 Kakoune 和 Vim 的启发，采用选区优先的编辑模型，并内置 tree-sitter 语法高亮和语言服务器协议（LSP）支持——无需插件或配置，即可在终端中获得现代 IDE 般的体验。命令为 hx。

🚀 为什么最新版本很重要：Helix 正在积极开发中，频繁发布的新版本带来了新的语言支持、编辑器功能和性能改进。最新版本提供了更好的 LSP 集成、更多内置主题以及更精细的编辑命令。

⚡ Helix 的核心功能

🌲 内置 Tree-sitter

由 tree-sitter 语法驱动，提供快速、精准的语法高亮和结构化选区。

🧠 内置语言服务器

通过 LSP 开箱即用地支持自动补全、诊断信息和跳转到定义。

🎯 选区优先编辑

多重选区模型，让复杂编辑变得直观且可重复。

⚙️ 零配置

合理的默认设置和内置主题——安装后即可立即投入使用。

🎯 为什么选择 Helix？

  • 开箱即用：无需插件配置即可获得语法高亮和 LSP 支持
  • 快速轻量：使用 Rust 编写，单一二进制文件，处理大文件依然流畅
  • 模态化人体工程学：精炼的“先选区后操作”编辑模型
  • 出色的默认配置：易于发现的按键映射和交互式命令面板

🆕 亮点功能

  • 内置 LSP：无需外部插件即可获得 IDE 功能
  • 多重选区：轻松同时编辑多处内容
  • 主题：内置数十种配色主题
  • 单一二进制文件：无需管理运行时依赖

❓ 常见问题

Helix 在官方 Ubuntu 软件源中吗？

不在 — 官方 Ubuntu 源没有打包 Helix。本免费镜像提供发布至少 2 个月的最新 Helix 版本；主仓库 deb.griffo.io 在上游发布后数小时内提供每个版本。

如何在 Ubuntu 上安装最新版 Helix？

按上方说明一次性添加 deb-free.griffo.io 仓库，然后运行：sudo apt install helix。之后通过常规的 sudo apt upgrade 获取新版本。

软件包是否有签名？如何构建？

每个软件包都使用仓库的 GPG 密钥（EA0F721D231FDD3A0A17B9AC7808B4DD62C41256）签名，并基于上游发布在公开的 GitHub 仓库中构建，任何人都可以审查。
