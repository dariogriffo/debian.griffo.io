什么是 Ruff？

Ruff 是一款极速的 Python linter 和代码格式化工具，由 uv 背后的团队 Astral 使用 Rust 编写。凭借 800 多条内置规则和对流行插件的原生重新实现，单个 ruff 二进制文件即可替代 Flake8、isort、Black、pyupgrade、autoflake 以及数十个插件，同时运行速度快 10–100 倍。它已成为现代 Python 生态的默认 linter，被 FastAPI、Pandas 和 Hugging Face 等项目采用。

😤 Python 工具链的痛点：要管理 flake8 外加一打插件、isort 和 Black 互相冲突、CI 中动辄数分钟的 lint 运行，还得在每一个 virtualenv 里安装 linter？Ruff 是一个零依赖的二进制文件，毫秒之间搞定这一切——用 apt 安装后即可在系统级使用，覆盖每个项目和 venv，无需 pip。
🚀 永久免费，安全修复立即发布：Debian 和 Ubuntu 在发布时会冻结软件包版本——本免费镜像提供发布至少 2 个月的最新版本，永久免费；安全修复立即发布。如需在上游发布后数小时内获取每个版本，请使用主仓库 deb.griffo.io。只需 apt upgrade 即可获取免费通道内的最新构建。

⚡ Ruff 的主要特性

🏎️ 快 10–100 倍

在传统工具需要数分钟的大型代码库上，Ruff 只需毫秒即可完成 lint——快到可以在每次保存和每条 CI 流水线中运行。

🧰 一个工具，800+ 规则

重新实现了 Flake8、isort、pyupgrade、pydocstyle、eradicate 等——pyproject.toml 中的一份配置即可替代一整套插件。

🖤 即插即用的格式化工具

ruff format 是与 Black 兼容的格式化工具——风格相同，速度快上几个数量级，无需安装额外工具。

🔧 自动修复

数百条规则支持自动修复：未使用的导入、语法升级、导入排序——一句 ruff check --fix 就搞定。

🐍 版本感知

支持 Python 3.7 及以上的任意版本，提供按版本区分的规则行为、适合 monorepo 的层级化配置以及按文件覆盖。

🔌 编辑器集成

内置语言服务器（ruff server），同一个二进制文件即可为 VS Code、Zed、Neovim 和 Helix 提供一流集成。

🏆 为什么用 apt 安装？

  • 系统级、无需 pip：一个二进制文件服务每个项目、venv、CI 任务和用户——无需维护各项目单独安装的版本
  • 非常适合 Docker/CI 镜像：将仓库预置到基础镜像中，每条流水线都能通过 apt upgrade 获取免费通道内的最新构建
  • 无需工具链：不需要 Rust、cargo 或 pip——一个签名的 apt 仓库，提供 8 种架构的预编译二进制文件
  • 自动更新：通过 apt upgrade 自动获取免费通道的最新构建
  • 与 uv 搭配：Astral 的包管理器也在这里提供——从一个仓库获取完整的现代 Python 工具链
