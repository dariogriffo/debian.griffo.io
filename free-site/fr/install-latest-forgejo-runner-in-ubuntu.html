Qu'est-ce que Forgejo Runner ?

Forgejo Runner est le runner CI pour Forgejo Actions. C'est un démon léger qui se connecte à votre instance Forgejo, récupère les tâches de workflow en attente et les exécute sur votre propre infrastructure — via Docker, LXC ou directement sur l'hôte — pour vous offrir un pipeline CI/CD entièrement auto-hébergé et compatible GitHub Actions.

🚀 Pourquoi les dernières versions sont importantes : Le runner suit de près les versions de Forgejo, en ajoutant la prise en charge de nouvelles fonctionnalités de workflow, une meilleure isolation et des corrections de bugs. Le maintenir à jour garantit la compatibilité avec les dernières capacités de Forgejo Actions.

⚡ Fonctionnalités clés de Forgejo Runner

🔁 Compatible Actions

Exécute les workflows en utilisant la même syntaxe et le même écosystème d'actions que GitHub Actions.

🐳 Exécution flexible

Exécutez les tâches dans des conteneurs Docker, en LXC, ou directement sur la machine hôte.

🔐 Auto-hébergé

Gardez votre CI/CD entièrement sur votre propre infrastructure, sans dépendance à un tiers.

⚖️ Évolutif

Enregistrez plusieurs runners avec des étiquettes pour répartir et cibler les charges de travail.

🎯 Pourquoi Forgejo Runner ?

  • Maîtrisez votre CI : Exécutez vos pipelines sur du matériel que vous contrôlez
  • Écosystème Actions : Réutilisez les définitions de workflow existantes de style GitHub
  • Isolation flexible : Exécution via Docker, LXC ou sur l'hôte selon vos besoins
  • Léger : Un seul binaire Go simple à déployer

🆕 Points forts

  • Exécution des tâches : Exécute les workflows Forgejo Actions en attente
  • Étiquettes : Ciblez des tâches vers des runners spécifiques
  • Support des conteneurs : Builds isolés via Docker
  • Mode démon : Fonctionne en continu en tant que service

❓ Questions fréquentes

Forgejo Runner est-il dans les dépôts officiels de Ubuntu ?

Non — Forgejo Runner n'est pas empaqueté dans les archives officielles de Ubuntu. Ce miroir gratuit propose la version la plus récente de Forgejo Runner datant d'au moins 2 mois ; le dépôt principal sur deb.griffo.io propose chaque version quelques heures après l'upstream.

Comment installer le dernier Forgejo Runner sur Ubuntu ?

Ajoutez le dépôt deb.griffo.io une seule fois (instructions ci-dessus), puis exécutez : sudo apt install forgejo-runner. Les nouvelles versions arrivent via le sudo apt upgrade habituel.

Les paquets sont-ils signés et comment sont-ils construits ?

Chaque paquet est signé avec la clé GPG du dépôt (EA0F721D231FDD3A0A17B9AC7808B4DD62C41256) et construit à partir des versions upstream dans des dépôts GitHub publics que chacun peut inspecter.

Quelles versions de Ubuntu sont prises en charge ?

Ubuntu 22.04 Jammy, 24.04 Noble, 25.10 Questing et 26.04 Resolute.
