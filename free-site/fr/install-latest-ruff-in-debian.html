Qu'est-ce que Ruff ?

Ruff est un linter et formateur de code Python ultra-rapide, écrit en Rust par Astral — l'équipe derrière uv. Avec plus de 800 règles intégrées et des réimplémentations natives des plugins populaires, un unique binaire ruff remplace Flake8, isort, Black, pyupgrade, autoflake et des dizaines de plugins, tout en s'exécutant 10 à 100× plus vite. Il est devenu le linter par défaut de l'écosystème Python moderne, adopté par des projets comme FastAPI, Pandas et Hugging Face.

😤 Les galères de l'outillage Python : Gérer flake8 et une douzaine de plugins, isort qui se bat avec Black, des lints de plusieurs minutes en CI, et des linters à installer dans chaque virtualenv ? Ruff est un binaire unique sans dépendances qui fait tout cela en quelques millisecondes — et l'installer avec apt le rend disponible sur tout le système, pour chaque projet et chaque venv, sans pip.
🚀 Gratuit pour toujours : Debian et Ubuntu figent les versions des paquets à chaque publication — ce miroir gratuit propose la version la plus récente de chaque paquet dès qu'elle date d'au moins 2 mois, avec des correctifs de sécurité immédiats. Le dépôt principal sur deb.griffo.io propose chaque version quelques heures après l'upstream.

⚡ Fonctionnalités clés de Ruff

🏎️ 10–100× plus rapide

Analyse de grandes bases de code en millisecondes là où les outils historiques prennent des minutes — assez rapide pour tourner à chaque sauvegarde et dans chaque pipeline CI.

🧰 Un outil, 800+ règles

Réimplémente Flake8, isort, pyupgrade, pydocstyle, eradicate et plus encore — une seule configuration dans pyproject.toml remplace une pile de plugins.

🖤 Formateur drop-in

ruff format est un formateur compatible Black — même style, plusieurs ordres de grandeur plus rapide, aucun outil séparé à installer.

🔧 Corrections automatiques

Des centaines de règles offrent des corrections automatiques : imports inutilisés, syntaxe modernisée, imports triés — ruff check --fix et c'est réglé.

🐍 Conscient des versions

Cible n'importe quel Python à partir de 3.7 avec un comportement des règles par version, une configuration hiérarchique adaptée aux monorepos et des exceptions par fichier.

🔌 Intégrations éditeur

Un serveur de langage intégré (ruff server) alimente des intégrations de premier ordre pour VS Code, Zed, Neovim et Helix depuis le même binaire.

🏆 Pourquoi l'installer avec apt ?

  • Système, sans pip : un binaire pour chaque projet, venv, job CI et utilisateur — pas d'installations par projet à garder synchronisées
  • Parfait pour les images Docker/CI : intégrez le dépôt dans vos images de base et chaque pipeline obtient automatiquement un ruff à jour
  • Aucune toolchain : pas de Rust, pas de cargo, pas de pip — un dépôt apt signé avec des binaires précompilés pour 8 architectures
  • Mises à jour automatiques : ruff publie fréquemment ; apt upgrade suit le rythme pour que vos règles ne soient jamais périmées
  • En tandem avec uv : le gestionnaire de paquets d'Astral est également disponible ici — toute la chaîne Python moderne depuis un seul dépôt

❓ Questions fréquentes

Ruff est-il dans les dépôts officiels de Debian ?

Oui — l'archive officielle de Debian fournit Ruff 0.0.291. Ce miroir gratuit propose la version la plus récente de Ruff datant d'au moins 2 mois, gratuitement pour toujours ; le dépôt principal sur deb.griffo.io propose chaque version quelques heures après l'upstream.

Comment installer le dernier Ruff sur Debian ?

Ajoutez le dépôt deb-free.griffo.io une seule fois (instructions ci-dessus), puis exécutez : sudo apt install ruff. Les nouvelles versions arrivent via le sudo apt upgrade habituel.

Les paquets sont-ils signés et comment sont-ils construits ?

Chaque paquet est signé avec la clé GPG du dépôt (EA0F721D231FDD3A0A17B9AC7808B4DD62C41256) et construit à partir des versions upstream dans des dépôts GitHub publics que chacun peut inspecter.

Quelles versions de Debian sont prises en charge ?

Debian 12 Bookworm, Debian 13 Trixie, Forky et Sid.
