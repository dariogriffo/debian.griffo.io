Qu'est-ce que Garage ?

Garage est un service de stockage objet distribué léger et compatible S3. Conçu pour fonctionner sur plusieurs machines réparties géographiquement, il fournit un stockage objet auto-hébergé et résilient avec une API compatible S3, ce qui en fait une solide alternative open source aux services de stockage objet hébergés pour les sauvegardes, les sites statiques et les données d'application.

🚀 Pourquoi les dernières versions sont importantes : Garage est activement développé, avec des améliorations de la réplication, des performances et de la couverture de l'API S3. Utiliser la dernière version vous donne accès aux fonctionnalités de stockage les plus récentes et aux correctifs de fiabilité.

⚡ Fonctionnalités clés de Garage

☁️ API compatible S3

Compatibilité directe avec les clients, SDK et outils S3 existants.

🌐 Géo-distribué

Répliquez les données sur plusieurs nœuds et sites pour plus de résilience.

🪶 Léger

Implémentation efficace en Rust qui fonctionne bien sur du matériel modeste.

🔒 Auto-hébergé

Gardez la pleine propriété de vos données sur votre propre infrastructure.

🎯 Pourquoi Garage ?

  • Maîtrisez votre stockage : Stockage objet auto-hébergé sur votre propre matériel
  • Écosystème S3 : Fonctionne avec le vaste éventail d'outils S3 existants
  • Résilient : La réplication multi-nœuds survit aux pannes de nœuds
  • Efficace : Faible empreinte en ressources, idéal pour les petits clusters

🆕 Points forts

  • Stockage objet : Buckets et objets compatibles S3
  • Réplication : Redondance configurable entre les nœuds
  • Multi-datacenter : Prise en compte des zones pour le placement
  • Hébergement statique : Servez des sites web statiques directement depuis les buckets

❓ Questions fréquentes

Garage est-il dans les dépôts officiels de Ubuntu ?

Non — Garage n'est pas empaqueté dans les archives officielles de Ubuntu. Ce miroir gratuit propose la version la plus récente de Garage datant d'au moins 2 mois ; le dépôt principal sur deb.griffo.io propose chaque version quelques heures après l'upstream.

Comment installer le dernier Garage sur Ubuntu ?

Ajoutez le dépôt deb-free.griffo.io une seule fois (instructions ci-dessus), puis exécutez : sudo apt install garage. Les nouvelles versions arrivent via le sudo apt upgrade habituel.

Les paquets sont-ils signés et comment sont-ils construits ?

Chaque paquet est signé avec la clé GPG du dépôt (EA0F721D231FDD3A0A17B9AC7808B4DD62C41256) et construit à partir des versions upstream dans des dépôts GitHub publics que chacun peut inspecter.

Quelles versions de Ubuntu sont prises en charge ?

Ubuntu 22.04 Jammy, 24.04 Noble, 25.10 Questing et 26.04 Resolute.
