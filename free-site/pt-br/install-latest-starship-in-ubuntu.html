O que é o Starship?

O Starship é um prompt rápido e personalizável para múltiplos shells escrito em Rust. Ele funciona com Bash, Zsh, Fish, PowerShell e mais, exibindo informações contextuais — status do git, versões de linguagens, contexto de nuvem e muito mais — através de um prompt único, consistente e altamente configurável que permanece ágil mesmo em repositórios grandes.

🚀 Por que as últimas versões são importantes: O Starship é desenvolvido ativamente com novos módulos, integrações e melhorias de desempenho. Executar a versão mais recente garante os módulos de prompt mais novos e opções de configuração adicionais.

⚡ Principais Recursos do Starship

⚡ Rápido

Escrito em Rust e otimizado para manter seu prompt responsivo em qualquer lugar.

🐚 Multi-shell

Um único prompt para Bash, Zsh, Fish, PowerShell, Nushell e outros.

🎨 Totalmente Customizável

Configure cada módulo e símbolo através de um único arquivo TOML.

🧩 Consciente do Contexto

Mostra estado do git, versões de linguagens, contexto de nuvem e mais, apenas quando relevante.

🎯 Por que o Starship?

  • Consistente em todo lugar: O mesmo prompt em todos os seus shells e máquinas
  • Contexto rico: Git, linguagens e ambiente em um relance
  • Altamente configurável: Ajuste cada detalhe via starship.toml
  • Rápido: Latência mínima mesmo em repositórios grandes

🆕 Destaques

  • Multi-shell: Funciona em todos os shells populares
  • Módulos: Dezenas de módulos de contexto incluídos
  • Configuração TOML: Arquivo de configuração único e portátil
  • Presets: Visuais prontos, como símbolos Nerd Font

❓ Perguntas frequentes

O Starship está nos repositórios oficiais do Ubuntu?

Sim — o arquivo oficial do Ubuntu traz Starship 1.22.1. Este espelho gratuito oferece a versão mais recente do Starship com pelo menos 2 meses, grátis para sempre; o repositório principal em deb.griffo.io oferece cada versão poucas horas após o upstream.

Como instalo o Starship mais recente no Ubuntu?

Adicione o repositório deb-free.griffo.io uma única vez com as instruções acima e execute: sudo apt install starship. Novos lançamentos chegam pelo sudo apt upgrade normal.

Os pacotes são assinados e como são construídos?

Cada pacote é assinado com a chave GPG do repositório (EA0F721D231FDD3A0A17B9AC7808B4DD62C41256) e construído a partir de lançamentos upstream em repositórios públicos do GitHub que qualquer pessoa pode inspecionar.

Quais versões do Ubuntu são suportadas?

Ubuntu 22.04 Jammy, 24.04 Noble, 25.10 Questing e 26.04 Resolute.
