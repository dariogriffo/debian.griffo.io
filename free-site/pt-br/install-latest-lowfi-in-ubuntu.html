O que é o lowfi?

lowfi é um pequeno aplicativo Rust com um único propósito: tocar lofi. Criado por talwat, ele faz isso da maneira mais simples possível - sem álbuns, sem anúncios, sem distrações, apenas música lofi pura para programar e focar. Foi projetado com a filosofia "suckless" em mente, fornecendo exatamente o que você precisa e nada mais.

🎯 Filosofia de Design: "Eu realmente odeio plataformas de música modernas, e queria um app pequeno e 'suckless' que apenas tocasse lofi aleatório sem vídeo." - talwat, criador do lowfi
🚀 Por que as versões mais recentes importam: lowfi é desenvolvido ativamente com lançamentos frequentes contendo novas fontes de faixas, melhorias de desempenho e recursos aprimorados. As versões mais recentes incluem suporte a listas de faixas personalizadas, integração MPRIS e melhor resiliência de rede para sessões de codificação ininterruptas.

⚡ Principais Recursos do lowfi

🎵 Simplicidade Pura

Aplicativo com propósito único - apenas toca música lofi. Sem interfaces complexas, sem distrações, perfeito para manter o estado de fluxo na programação.

🌐 Resiliente à Rede

Armazena 5 músicas inteiras em buffer ao invés de fragmentos de streaming. Projetado para redes instáveis e audição ininterrupta.

🎛️ Suporte a Teclas de Mídia

Funciona com teclas de mídia e MPRIS (playerctl). Controle a reprodução sem sair do editor ou interromper o foco.

📻 Listas de Faixas Personalizadas

Suporte a listas de faixas personalizadas incluindo Chillhop, arquivos locais e playlists da comunidade para variedade.

⚡ Leve

Binário pequeno com uso mínimo de recursos. Não vai diminuir seu ambiente de desenvolvimento ou consumir recursos do sistema.

🎨 Interface Personalizável

Modo minimalista, exibição sem bordas, suporte a tela alternativa e largura do player ajustável para diferentes configurações de terminal.

❓ Perguntas frequentes

O lowfi está nos repositórios oficiais do Ubuntu?

Não — lowfi não está empacotado nos arquivos oficiais do Ubuntu. Este espelho gratuito oferece a versão mais recente do lowfi com pelo menos 2 meses; o repositório principal em deb.griffo.io oferece cada versão poucas horas após o upstream.

Como instalo o lowfi mais recente no Ubuntu?

Adicione o repositório deb-free.griffo.io uma única vez com as instruções acima e execute: sudo apt install lowfi. Novos lançamentos chegam pelo sudo apt upgrade normal.

Os pacotes são assinados e como são construídos?

Cada pacote é assinado com a chave GPG do repositório (EA0F721D231FDD3A0A17B9AC7808B4DD62C41256) e construído a partir de lançamentos upstream em repositórios públicos do GitHub que qualquer pessoa pode inspecionar.

Quais versões do Ubuntu são suportadas?

Ubuntu 22.04 Jammy, 24.04 Noble, 25.10 Questing e 26.04 Resolute.
