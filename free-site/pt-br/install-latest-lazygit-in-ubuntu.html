O que é o lazygit?

lazygit é uma interface de usuário de terminal simples para comandos git criada por Jesse Duffield. Ela transforma operações complexas do Git em interações visuais intuitivas, tornando o controle de versão acessível e eficiente para desenvolvedores de todos os níveis. Com mais de 63k estrelas no GitHub, tornou-se a TUI Git preferida de desenvolvedores no mundo todo.

😤 Problemas do Git Resolvidos: Cansado de editar arquivos TODO para rebase interativo? Frustrado com arquivos de patch para staging? Entediado com fluxos de trabalho Git complexos na linha de comando? lazygit torna as operações do Git visuais, intuitivas e até agradáveis de usar.
🚀 Por que as versões mais recentes importam: lazygit é desenvolvido ativamente com lançamentos frequentes contendo novos recursos, melhorias de desempenho e aprimoramentos de fluxo de trabalho. As versões mais recentes incluem rebase interativo aprimorado, melhor resolução de conflitos, comandos personalizados avançados e correções de bugs críticos.

⚡ Principais Recursos do lazygit

📝 Staging Visual

Faça staging de linhas ou hunks individuais com teclas simples. Sem mais edição de patches - basta pressionar espaço para selecionar o que deseja.

🔄 Rebase Interativo

Faça squash, fixup, drop, edit e reordene commits com atalhos de teclado intuitivos. Sem necessidade de editar arquivos TODO.

🍒 Cherry-picking

Copie e cole commits entre branches com shift+c e shift+v. Seleção e aplicação visual de commits.

🔍 Git Bisect

Interface visual de git bisect para encontrar commits problemáticos rapidamente. Marque commits como bons/ruins com teclas simples.

🌳 Worktrees

Crie e gerencie Git worktrees para trabalhar em múltiplos branches simultaneamente sem stashing ou commits WIP.

⚙️ Comandos Personalizados

Sistema extensível de comandos personalizados para adicionar seus próprios fluxos de trabalho Git e atalhos diretamente na interface.

🚀 Como os dois espelhos se relacionam: o repositório principal em deb.griffo.io publica novas versões upstream em poucas horas; este espelho gratuito serve cada versão quando ela tem pelo menos 2 meses — correções de segurança imediatamente. Um simples apt upgrade mantém você em dia dentro dessa janela.

🎯 Exemplos de uso básico

Iniciar o lazygit:

# Start lazygit in current Git repository lazygit # Create alias for convenience echo "alias lg='lazygit'" >> ~/.bashrc source ~/.bashrc # Now use the alias lg

Atalhos de teclado essenciais:

# Navigation j/k or ↑/↓ - Move up/down h/l or ←/→ - Switch between panels tab - Next panel shift+tab - Previous panel # Staging space - Stage/unstage file or hunk a - Stage all changes in file v - Start visual selection # Commits c - Commit staged changes shift+c - Copy commit (cherry-pick) shift+v - Paste commit # Rebasing i - Start interactive rebase s - Squash commit f - Fixup commit d - Drop commit

Fluxos de trabalho avançados:

# Undo/Redo z - Undo last action ctrl+z - Redo last undone action # Worktrees w - Create worktree from branch # Filtering / - Filter current view # Custom commands : - Open custom command prompt

❓ Perguntas frequentes

O lazygit está nos repositórios oficiais do Ubuntu?

Sim — o arquivo oficial do Ubuntu traz lazygit 0.57.0. Este espelho gratuito oferece a versão mais recente do lazygit com pelo menos 2 meses, grátis para sempre; o repositório principal em deb.griffo.io oferece cada versão poucas horas após o upstream.

Como instalo o lazygit mais recente no Ubuntu?

Adicione o repositório deb-free.griffo.io uma única vez com as instruções acima e execute: sudo apt install lazygit. Novos lançamentos chegam pelo sudo apt upgrade normal.

Os pacotes são assinados e como são construídos?

Cada pacote é assinado com a chave GPG do repositório (EA0F721D231FDD3A0A17B9AC7808B4DD62C41256) e construído a partir de lançamentos upstream em repositórios públicos do GitHub que qualquer pessoa pode inspecionar.

Quais versões do Ubuntu são suportadas?

Ubuntu 22.04 Jammy, 24.04 Noble, 25.10 Questing e 26.04 Resolute.
