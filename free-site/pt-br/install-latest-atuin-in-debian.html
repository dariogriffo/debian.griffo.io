O que é o Atuin?

O Atuin substitui o histórico do seu shell por um banco de dados SQLite pesquisável, adicionando busca full-text poderosa, sincronização opcional com criptografia ponta a ponta entre máquinas, e estatísticas úteis. Ele registra contexto adicional para cada comando — código de saída, duração, diretório e mais — e funciona com Bash, Zsh, Fish e outros shells.

🚀 Por que as últimas versões são importantes: O Atuin é desenvolvido ativamente com melhorias em busca, sincronização e na interface interativa. Executar a versão mais recente garante os recursos de histórico mais novos e correções.

⚡ Principais Recursos do Atuin

🔍 Busca Poderosa

Busca full-text, fuzzy e por prefixo em todo o seu histórico de shell.

🔐 Sincronização Criptografada

Sincronização opcional com criptografia ponta a ponta do histórico entre todas as suas máquinas.

📊 Contexto Rico

Registra código de saída, duração, diretório e mais para cada comando.

🐚 Multi-shell

Funciona com Bash, Zsh, Fish e outros shells populares.

🎯 Por que o Atuin?

  • Histórico melhor: Histórico pesquisável em SQLite que nunca trunca silenciosamente
  • Sincronização entre máquinas: Sincronização com criptografia ponta a ponta, auto-hospedável
  • Rico em contexto: Saiba quando, onde e como cada comando foi executado
  • Estatísticas: Insights sobre seus comandos mais usados

🆕 Destaques

  • Histórico em SQLite: Histórico de comandos durável e consultável
  • Busca Interativa: Busca fuzzy vinculada à sua seta para cima
  • Sincronização: Sincronização criptografada opcional entre múltiplas máquinas
  • Estatísticas: Estatísticas de uso de comandos

❓ Perguntas frequentes

O Atuin está nos repositórios oficiais do Debian?

Sim — o arquivo oficial do Debian traz Atuin 18.8.0. Este espelho gratuito oferece a versão mais recente do Atuin com pelo menos 2 meses, grátis para sempre; o repositório principal em deb.griffo.io oferece cada versão poucas horas após o upstream.

Como instalo o Atuin mais recente no Debian?

Adicione o repositório deb-free.griffo.io uma única vez com as instruções acima e execute: sudo apt install atuin. Novos lançamentos chegam pelo sudo apt upgrade normal.

Os pacotes são assinados e como são construídos?

Cada pacote é assinado com a chave GPG do repositório (EA0F721D231FDD3A0A17B9AC7808B4DD62C41256) e construído a partir de lançamentos upstream em repositórios públicos do GitHub que qualquer pessoa pode inspecionar.

Quais versões do Debian são suportadas?

Debian 12 Bookworm, Debian 13 Trixie, Forky e Sid.
