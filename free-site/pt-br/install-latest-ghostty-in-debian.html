O que é o Ghostty?

O Ghostty é um emulador de terminal rápido, rico em recursos e multiplataforma que usa interface nativa da plataforma e aceleração por GPU. Criado por Mitchell Hashimoto (fundador da HashiCorp), o Ghostty se diferencia por oferecer velocidade, recursos e experiência nativa sem forçar você a escolher entre eles.

As atualizações do Ghostty estarão disponíveis apenas no Trixie e versões mais recentes do Debian. O Bookworm não será atualizado além da versão 1.1.3

⚡ Principais Recursos do Ghostty

🎮 Aceleração por GPU

Usa OpenGL no Linux e Metal no macOS para renderização acelerada por hardware. Mantém 60fps sob carga pesada com desempenho superior.

🖥️ Interface Nativa

Experiência verdadeiramente nativa da plataforma. SwiftUI no macOS, GTK no Linux. Janelas reais, barras de menu e painéis de configurações nativos.

📊 Conforme com Padrões

Um dos emuladores de terminal mais conformes disponíveis. Compatibilidade abrangente com xterm e inovações modernas de terminal.

🔧 Recursos Avançados

Suporte a múltiplas janelas, abas, divisões, ligaduras e opções extensas de personalização sem sacrificar o desempenho.

🏆 Benchmarks de Desempenho

  • 4x mais rápido que iTerm e Kitty para renderização de texto grande
  • 2x mais rápido que Terminal.app para operações de IO
  • Competitivo com Alacritty enquanto oferece recursos muito mais ricos
  • Renderização a 60fps mantida sob carga pesada
  • IO com baixa latência com thread de IO dedicada

❓ Perguntas frequentes

O Ghostty está nos repositórios oficiais do Debian?

Não — Ghostty não está empacotado nos arquivos oficiais do Debian. Este espelho gratuito oferece a versão mais recente do Ghostty com pelo menos 2 meses; o repositório principal em deb.griffo.io oferece cada versão poucas horas após o upstream.

Como instalo o Ghostty mais recente no Debian?

Adicione o repositório deb-free.griffo.io uma única vez com as instruções acima e execute: sudo apt install ghostty. Novos lançamentos chegam pelo sudo apt upgrade normal.

Os pacotes são assinados e como são construídos?

Cada pacote é assinado com a chave GPG do repositório (EA0F721D231FDD3A0A17B9AC7808B4DD62C41256) e construído a partir de lançamentos upstream em repositórios públicos do GitHub que qualquer pessoa pode inspecionar.

Quais versões do Debian são suportadas?

Debian 12 Bookworm, Debian 13 Trixie, Forky e Sid.
