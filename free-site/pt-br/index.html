⚡ Instalação rápida

Adicione este repositório ao seu sistema Debian e comece a instalar as versões mais recentes das suas ferramentas de desenvolvimento favoritas:

Passo 1: Adicionar repositório

sudo install -d -m 0755 /etc/apt/keyrings curl -fsSL https://deb-free.griffo.io/EA0F721D231FDD3A0A17B9AC7808B4DD62C41256.asc | sudo gpg --dearmor --yes -o /etc/apt/keyrings/deb-free.griffo.io.gpg echo "deb [signed-by=/etc/apt/keyrings/deb-free.griffo.io.gpg] https://deb-free.griffo.io/apt $(lsb_release -sc 2>/dev/null) main" | sudo tee /etc/apt/sources.list.d/deb-free.griffo.io.list > /dev/null sudo apt update

🚀 Por que escolher deb.griffo.io?

Este repositório não oficial se especializa em fornecer a versão mais recente de cada ferramenta de desenvolvimento popular, com pelo menos 2 meses de atraso do upstream, como pacotes Debian corretamente empacotados — grátis para sempre. Enquanto os repositórios oficiais do Debian frequentemente ficam anos atrás dos lançamentos upstream ou nem chegam a empacotar a ferramenta, garantimos que você obtenha:

  • ✅ Versões estáveis com no máximo 2 meses de atraso do upstream, grátis para sempre
  • ✅ Correções de segurança publicadas imediatamente
  • ✅ Builds noturnos para recursos de ponta (Zig, ZLS)
  • ✅ Empacotamento Debian adequado com dependências corretas
  • ✅ Suporte a múltiplas distribuições (Bookworm, Trixie, Forky, Sid)
  • ✅ Sem conta, sem autenticação — e, se precisar de cada versão poucas horas após o upstream, o repositório principal em deb.griffo.io está disponível

⚠️ Informações importantes

Escopo: Este repositório se concentra exclusivamente em empacotamento não oficial para Debian. Para problemas com as ferramentas em si, entre em contato com os respectivos projetos originais.

Atualizações: Os pacotes são atualizados automaticamente seguindo a janela de pelo menos 2 meses deste espelho gratuito; correções de segurança são publicadas imediatamente. Para cada versão poucas horas após o upstream, use o repositório principal em deb.griffo.io.

Comunidade: Este é um projeto impulsionado pela comunidade destinado a preencher a lacuna entre as ferramentas de desenvolvimento em rápida evolução e o ciclo de empacotamento estável do Debian.
