O que é o Helix?

O Helix é um editor de texto modal pós-moderno escrito em Rust. Inspirado no Kakoune e no Vim, ele usa um modelo de edição baseado em seleção e vem com destaque de sintaxe via tree-sitter e suporte ao Language Server Protocol prontos para uso — sem necessidade de plugins ou configuração para obter uma experiência moderna semelhante a uma IDE no seu terminal. O comando é hx.

🚀 Por que as últimas versões são importantes: O Helix está em desenvolvimento ativo, com lançamentos frequentes que adicionam suporte a novas linguagens, recursos do editor e melhorias de desempenho. As versões mais recentes trazem melhor integração com LSP, mais temas embutidos e comandos de edição refinados.

⚡ Principais Recursos do Helix

🌲 Tree-sitter Integrado

Destaque de sintaxe rápido e preciso, além de seleções estruturais baseadas em gramáticas tree-sitter.

🧠 Language Servers Incluídos

Autocompletar, diagnósticos e ir para definição funcionam prontos para uso via LSP.

🎯 Edição Baseada em Seleção

Um modelo de múltiplas seleções que torna edições complexas intuitivas e repetíveis.

⚙️ Zero Configuração

Padrões sensatos e temas embutidos — produtivo no momento em que você o instala.

  • Rápido e leve: Escrito em Rust, binário único, ágil em arquivos grandes

  • Ótimos padrões: Mapas de teclas fáceis de descobrir e uma paleta de comandos interativa

❓ Perguntas frequentes

O Helix está nos repositórios oficiais do Ubuntu?

Não — Helix não está empacotado nos arquivos oficiais do Ubuntu. Este espelho gratuito oferece a versão mais recente do Helix com pelo menos 2 meses; o repositório principal em deb.griffo.io oferece cada versão poucas horas após o upstream.

Como instalo o Helix mais recente no Ubuntu?

Adicione o repositório deb.griffo.io uma única vez com as instruções acima e execute: sudo apt install helix. Novos lançamentos chegam pelo sudo apt upgrade normal.

Os pacotes são assinados e como são construídos?

Cada pacote é assinado com a chave GPG do repositório (EA0F721D231FDD3A0A17B9AC7808B4DD62C41256) e construído a partir de lançamentos upstream em repositórios públicos do GitHub que qualquer pessoa pode inspecionar.

Quais versões do Ubuntu são suportadas?

Ubuntu 22.04 Jammy, 24.04 Noble, 25.10 Questing e 26.04 Resolute.
