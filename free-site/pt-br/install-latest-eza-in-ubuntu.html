O que é o eza?

O eza é um substituto moderno e mantido para o clássico programa de listagem de arquivos em linha de comando ls, incluído nos sistemas Unix e Linux. Escrito em Rust, o eza oferece mais recursos e melhores padrões ao mesmo tempo que é pequeno, rápido e apenas um único binário.

🚀 Por que as últimas versões importam: O eza é desenvolvido ativamente com lançamentos frequentes contendo novos recursos, melhorias de desempenho e correções de bugs. As últimas versões incluem suporte a hiperlinks, temas personalizados, integração Git aprimorada e muitas melhorias de usabilidade em relação ao projeto exa original.

⚡ Principais Recursos do eza

🎨 Saída Visual Rica

Cores para distinguir tipos de arquivos, suporte a ícones e temas personalizáveis. Torna a navegação de arquivos mais intuitiva e visualmente atraente.

🔗 Integração com Git

Exibe o status do Git para arquivos e repositórios, rastreia arquivos ignorados e mostra informações do repositório diretamente nas listagens.

📊 Metadados Estendidos

Exibe links simbólicos, atributos estendidos, pontos de montagem, contexto SELinux e datas relativas legíveis por humanos.

🔧 Altamente Configurável

Temas personalizados via theme.yml, opções extensas de linha de comando e compatibilidade retroativa com ls e exa.

🆚 eza vs ls vs exa

  • vs ls: Cores, ícones, integração Git, visualização em árvore, melhores padrões
  • vs exa: Mantido ativamente, corrige o "Bug de Grade", hiperlinks, temas, correções de segurança
  • Desempenho: Implementação rápida em Rust, binário único, dependências mínimas
  • Compatibilidade: Substituto direto com opções familiares
🚀 Sempre à frente dos repositórios oficiais: Debian e Ubuntu congelam as versões dos pacotes a cada lançamento — este repositório publica novas versões upstream normalmente em poucas horas. Um simples apt upgrade mantém você sempre na versão mais recente.

🎯 Exemplos de uso básico

Substituir ls pelo eza:

# Listagem básica com cores e ícones eza # Formato longo com detalhes eza -l # Visualização em árvore eza --tree # Exibir status do Git eza -l --git # Visualização em grade com ícones eza --icons --grid

Recursos avançados:

# Mostrar todos os arquivos incluindo ocultos eza -la # Ordenar por tamanho, maior primeiro eza -l --sort=size --reverse # Mostrar apenas diretórios eza -D # Listagem recursiva com limite de profundidade eza -R --level=2 # Formato de hora personalizado eza -l --time-style=iso

Criar alias para uso permanente:

# Adicionar a ~/.bashrc ou ~/.zshrc echo 'alias ls="eza"' >> ~/.bashrc echo 'alias ll="eza -l"' >> ~/.bashrc echo 'alias la="eza -la"' >> ~/.bashrc # Recarregar o shell source ~/.bashrc

❓ Perguntas frequentes

O eza está nos repositórios oficiais do Ubuntu?

Sim — o arquivo oficial do Ubuntu traz eza 0.23.4. Este espelho gratuito oferece a versão mais recente do eza com pelo menos 2 meses, grátis para sempre; o repositório principal em deb.griffo.io oferece cada versão poucas horas após o upstream.

Como instalo o eza mais recente no Ubuntu?

Adicione o repositório deb.griffo.io uma única vez com as instruções acima e execute: sudo apt install eza. Novos lançamentos chegam pelo sudo apt upgrade normal.

Os pacotes são assinados e como são construídos?

Cada pacote é assinado com a chave GPG do repositório (EA0F721D231FDD3A0A17B9AC7808B4DD62C41256) e construído a partir de lançamentos upstream em repositórios públicos do GitHub que qualquer pessoa pode inspecionar.

Quais versões do Ubuntu são suportadas?

Ubuntu 22.04 Jammy, 24.04 Noble, 25.10 Questing e 26.04 Resolute.
