O que é o Ruff?

O Ruff é um linter e formatador de código Python extremamente rápido, escrito em Rust pela Astral — a equipe por trás do uv. Com mais de 800 regras integradas e reimplementações nativas de plugins populares, um único binário ruff substitui Flake8, isort, Black, pyupgrade, autoflake e dezenas de plugins, rodando de 10 a 100× mais rápido. Ele se tornou o linter padrão do ecossistema Python moderno, adotado por projetos como FastAPI, Pandas e Hugging Face.

😤 Dores do ferramental Python: Gerenciar o flake8 mais uma dúzia de plugins, o isort brigando com o Black, execuções de lint que levam minutos no CI, e instalar linters em cada virtualenv? O Ruff é um único binário sem dependências que faz tudo isso em milissegundos — e instalá-lo com o apt o torna disponível para todo o sistema, para cada projeto e venv, sem pip.
🚀 Sempre à frente dos repositórios oficiais: Debian e Ubuntu congelam as versões dos pacotes a cada lançamento — este repositório publica novas versões upstream normalmente em poucas horas. Um simples apt upgrade mantém você sempre na versão mais recente.

⚡ Principais Recursos do Ruff

🏎️ De 10 a 100× Mais Rápido

Analisa grandes bases de código em milissegundos onde ferramentas legadas levam minutos — rápido o suficiente para rodar a cada salvamento e em cada pipeline de CI.

🧰 Uma Ferramenta, Mais de 800 Regras

Reimplementa Flake8, isort, pyupgrade, pydocstyle, eradicate e mais — uma configuração no pyproject.toml substitui uma pilha de plugins.

🖤 Formatador Substituto Direto

ruff format é um formatador compatível com o Black — mesmo estilo, ordens de magnitude mais rápido, sem ferramenta separada para instalar.

🔧 Correções Automáticas

Centenas de regras vêm com correções automáticas: imports não utilizados, sintaxe atualizada, imports ordenados — ruff check --fix e pronto.

🐍 Ciente da Versão

Suporta qualquer Python a partir do 3.7 com comportamento de regras por versão, configuração hierárquica amigável a monorepos e substituições por arquivo.

🔌 Integrações com Editores

Um servidor de linguagem integrado (ruff server) alimenta integrações de primeira classe com VS Code, Zed, Neovim e Helix a partir do mesmo binário.

🏆 Por que instalar com o apt?

  • Para todo o sistema, sem pip: um binário para cada projeto, venv, job de CI e usuário — sem instalações por projeto para manter em sincronia
  • Perfeito para imagens Docker/CI: inclua o repositório nas imagens base e cada pipeline recebe o ruff atual automaticamente
  • Sem toolchain: sem Rust, sem cargo, sem pip — um repositório apt assinado com binários pré-compilados para 8 arquiteturas
  • Atualizações automáticas: o ruff lança versões com frequência; o apt upgrade acompanha para que suas regras nunca fiquem desatualizadas
  • Combina com o uv: o gerenciador de pacotes da Astral também é servido aqui — o toolchain Python moderno completo em um único repositório

❓ Perguntas frequentes

O Ruff está nos repositórios oficiais do Debian?

Sim — o arquivo oficial do Debian traz Ruff 0.0.291. Este espelho gratuito oferece a versão mais recente do Ruff com pelo menos 2 meses, grátis para sempre; o repositório principal em deb.griffo.io oferece cada versão poucas horas após o upstream.

Como instalo o Ruff mais recente no Debian?

Adicione o repositório deb-free.griffo.io uma única vez com as instruções acima e execute: sudo apt install ruff. Novos lançamentos chegam pelo sudo apt upgrade normal.

Os pacotes são assinados e como são construídos?

Cada pacote é assinado com a chave GPG do repositório (EA0F721D231FDD3A0A17B9AC7808B4DD62C41256) e construído a partir de lançamentos upstream em repositórios públicos do GitHub que qualquer pessoa pode inspecionar.

Quais versões do Debian são suportadas?

Debian 12 Bookworm, Debian 13 Trixie, Forky e Sid.
