O que é o Fastfetch?

O Fastfetch é uma ferramenta no estilo do neofetch para coletar informações do sistema e exibi-las de forma bonita no seu terminal — SO, kernel, hardware, tempo de atividade, pacotes, ambiente de desktop e muito mais, ao lado do logo da sua distro. Escrito principalmente em C com foco em desempenho e personalização, tornou-se o sucessor de fato quando o neofetch foi arquivado, e é desenvolvido ativamente com lançamentos frequentes.

😤 O problema da versão do arquivo: Versões mais antigas do Ubuntu não incluem o fastfetch, e onde o arquivo o tem, ele está congelado no momento da importação. O upstream avança rápido: novos logos, novos módulos de detecção e correções para novos hardwares chegam constantemente, então a versão do arquivo fica silenciosamente para trás. Este repositório fornece compilações upstream atuais mantendo exatamente o mesmo layout de arquivos do pacote oficial do arquivo, então a troca é transparente em ambas as direções.
🚀 Como os dois espelhos se relacionam: o repositório principal em deb.griffo.io publica novas versões upstream em poucas horas; este espelho gratuito serve cada versão quando ela tem pelo menos 2 meses — correções de segurança imediatamente. Um simples apt upgrade mantém você em dia dentro dessa janela.

⚡ Principais Recursos do Fastfetch

🏎️ Rápido por Design

Escrito em C com detecção concorrente — renderiza as informações completas do sistema em milissegundos, uma fração do tempo de inicialização do neofetch.

🧩 Mais de 50 Módulos de Informação

SO, kernel, CPU, GPU, memória, discos, bateria, tela, DE/WM, terminal, pacotes, reprodutor de mídia e muitos outros — cada um configurável individualmente.

🎨 Presets Incluídos

Este pacote traz todos os presets integrados em /usr/share/fastfetch/presets/ — incluindo um preset neofetch fiel para familiaridade instantânea.

📝 Configuração em JSONC

Um único arquivo de configuração JSONC documentado controla módulos, formatação e logos. Gere um ponto de partida com fastfetch --gen-config.

🖼️ Flexibilidade de Logos

Centenas de logos de distros integrados, além de logos em imagem via protocolos sixel/kitty/iTerm, ASCII art personalizada, ou nenhum logo.

🐚 Completions de Shell

Completions para bash, zsh e fish, além de uma página de manual — instalados nos caminhos padrão do Debian, correspondendo ao pacote do arquivo.

🏆 Por que este repositório em vez da versão do arquivo?

  • Versão atual: novos lançamentos upstream em poucas horas, sem congelar durante toda a vida útil de um lançamento do Debian
  • Mesmo layout: alinhado arquivo por arquivo com o pacote oficial do Debian — presets, completions e página de manual nos lugares padrão
  • Troca transparente: instalar a partir deste repositório simplesmente atualiza o pacote do arquivo; remover o repositório reverte de forma limpa
  • Atualizações automáticas: novos lançamentos chegam com o seu apt upgrade de sempre
  • Multi-arquitetura: amd64, arm64, armhf, ppc64el, s390x e riscv64

❓ Perguntas frequentes

O Fastfetch está nos repositórios oficiais do Ubuntu?

Sim — o arquivo oficial do Ubuntu traz Fastfetch 2.57.1. Este espelho gratuito oferece a versão mais recente do Fastfetch com pelo menos 2 meses, grátis para sempre; o repositório principal em deb.griffo.io oferece cada versão poucas horas após o upstream.

Como instalo o Fastfetch mais recente no Ubuntu?

Adicione o repositório deb-free.griffo.io uma única vez com as instruções acima e execute: sudo apt install fastfetch. Novos lançamentos chegam pelo sudo apt upgrade normal.

Os pacotes são assinados e como são construídos?

Cada pacote é assinado com a chave GPG do repositório (EA0F721D231FDD3A0A17B9AC7808B4DD62C41256) e construído a partir de lançamentos upstream em repositórios públicos do GitHub que qualquer pessoa pode inspecionar.

Quais versões do Ubuntu são suportadas?

Ubuntu 22.04 Jammy, 24.04 Noble, 25.10 Questing e 26.04 Resolute.
