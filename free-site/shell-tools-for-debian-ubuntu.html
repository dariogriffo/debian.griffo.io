for Debian & Ubuntu — free forever via apt

Faster, friendlier replacements for classic Unix utilities — search, navigation, shell history and prompts. Every tool below installs with one apt command from deb.griffo.io, is GPG-signed, and serves the newest release that is at least 2 months old — security fixes ship immediately; no snap, no manual builds, no account needed.
