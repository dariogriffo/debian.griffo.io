O que é o lazydocker?

O lazydocker é uma interface de terminal simples para Docker e docker-compose criada por Jesse Duffield (mesmo autor do lazygit). Ele fornece uma interface unificada para gerenciar seu ambiente Docker, visualizar logs, monitorar métricas e realizar operações comuns sem memorizar comandos complexos. Com mais de 45 mil estrelas no GitHub, é o TUI Docker mais popular disponível.

😤 Problemas do Docker Resolvidos: Cansado de gerenciar múltiplas janelas de terminal para diferentes contêineres? Farto de memorizar comandos docker? Frustrado com o rastreamento de estados de contêineres entre serviços? O lazydocker coloca tudo que você precisa em uma interface intuitiva.
🚀 Por que as últimas versões importam: O lazydocker é desenvolvido ativamente com lançamentos frequentes contendo suporte a novos recursos do Docker, melhorias de desempenho e capacidades aprimoradas de gerenciamento de contêineres. As últimas versões incluem melhor visualização de métricas, tratamento de logs aprimorado e integração docker-compose melhorada.

⚡ Principais Recursos do lazydocker

📊 Visão Geral Visual

Veja o estado de todo o seu ambiente Docker de relance. Contêineres, imagens, volumes e redes em uma visão unificada.

📈 Métricas em Tempo Real

Gráficos ASCII mostrando uso de CPU, memória e rede. Métricas personalizáveis para monitorar exatamente o que você precisa para seus contêineres.

📝 Gerenciamento de Logs

Visualize logs de contêineres e serviços com filtragem, acompanhamento e recursos de busca. Sem mais necessidade de múltiplas janelas de terminal.

🔧 Operações de Contêiner

Inicie, pare, reinicie, remova e reconstrua contêineres com um único pressionamento de tecla. Conecte-se a contêineres para depuração interativa.

🗂️ Gerenciamento de Imagens

Visualize camadas de imagens, ancestralidade e uso de disco. Limpe imagens, contêineres e volumes não utilizados para recuperar espaço em disco.

🐭 Suporte a Mouse

Suporte completo a mouse para clicar, rolar e selecionar. Atalhos de teclado para usuários avançados, mouse para conveniência.

🎯 Exemplos de uso básico

Iniciar o lazydocker:

# Iniciar o lazydocker lazydocker # Criar alias conveniente echo "alias lzd='lazydocker'" >> ~/.bashrc source ~/.bashrc # Usar o alias lzd

Atalhos de teclado essenciais:

# Navegação j/k ou ↑/↓ - Mover para cima/baixo h/l ou ←/→ - Alternar entre painéis tab - Próximo painel # Operações de contêiner r - Reiniciar contêiner s - Parar contêiner d - Remover contêiner e - Executar no contêiner m - Visualizar logs # Operações do sistema p - Limpar contêineres/imagens não utilizados R - Reconstruir serviço (docker-compose) u - Atualizar visualização
