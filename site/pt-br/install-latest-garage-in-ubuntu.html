O que é o Garage?

O Garage é um serviço de armazenamento de objetos distribuído, leve e compatível com S3. Projetado para rodar em várias máquinas geograficamente distribuídas, ele oferece armazenamento de objetos resiliente e auto-hospedado com uma API compatível com S3, tornando-se uma forte alternativa de código aberto aos serviços de armazenamento de objetos hospedados para backups, sites estáticos e dados de aplicações.

🚀 Por que as últimas versões são importantes: O Garage é desenvolvido ativamente com melhorias em replicação, desempenho e cobertura da API S3. Executar a versão mais recente garante os recursos de armazenamento mais novos e correções de confiabilidade.

⚡ Principais Recursos do Garage

☁️ API compatível com S3

Compatibilidade direta com clientes, SDKs e ferramentas S3 existentes.

🌐 Geodistribuído

Replique dados entre múltiplos nós e sites para maior resiliência.

🪶 Leve

Implementação eficiente em Rust que roda bem em hardware modesto.

🔒 Auto-hospedado

Mantenha total propriedade sobre seus dados na sua própria infraestrutura.

🎯 Por que o Garage?

  • Seu próprio armazenamento: Armazenamento de objetos auto-hospedado no seu próprio hardware
  • Ecossistema S3: Funciona com a vasta gama de ferramentas S3 existentes
  • Resiliente: Replicação entre múltiplos nós sobrevive a falhas de nós
  • Eficiente: Baixo consumo de recursos, ideal para clusters pequenos

🆕 Destaques

  • Armazenamento de Objetos: Buckets e objetos compatíveis com S3
  • Replicação: Redundância configurável entre nós
  • Multi-datacenter: Reconhecimento de zonas para posicionamento
  • Hospedagem Estática: Sirva sites estáticos diretamente dos buckets
