O que é o just?

O just é um prático executor de comandos escrito em Rust por Casey Rodarmor. Ele salva e executa comandos específicos de projetos, chamados receitas, armazenados em um arquivo chamado justfile com uma sintaxe inspirada no make. Ao contrário do make, o just não é um sistema de compilação: as receitas são simplesmente comandos a executar, então não há alvos fictícios nem dependência dos horários de modificação dos arquivos. Com mais de 25 mil estrelas no GitHub, ele se tornou o executor de tarefas preferido para projetos modernos.

😤 Problemas do make Tradicional: Cansado de declarações .PHONY por toda parte? Farto de erros de tabulação versus espaço e regras de compilação implícitas que disparam quando você menos espera? O just mantém a familiaridade das receitas de um Makefile, mas descarta a bagagem de sistema de compilação — toda receita é apenas um comando.
🚀 Por que as versões mais recentes importam: O just está sendo ativamente desenvolvido com lançamentos frequentes contendo novos atributos, funções de expressão, integrações com shell e melhorias de qualidade de vida. As versões mais recentes incluem scripts de autocompletar aprimorados, novas funções integradas, melhores mensagens de erro e suporte multiplataforma expandido.

⚡ Principais Recursos do just

📝 Sintaxe Inspirada no Make

Escreva receitas em um justfile familiar e legível. Se você conhece Makefiles, já conhece a maior parte do just — sem as esquisitices.

🧩 Parâmetros de Receita

As receitas aceitam argumentos e têm valores padrão, então uma única receita pode se adaptar a muitas invocações sem scripts extras.

🐚 Qualquer Shell ou Linguagem

Escreva receitas em sh, bash, Python, Node ou qualquer outra coisa com uma linha shebang. O just executa cada receita com o interpretador que você escolher.

📋 Listar e Descobrir

Execute just --list para ver todas as receitas com seus comentários de documentação. Novatos podem explorar as tarefas de um projeto instantaneamente.

🔐 .env e Variáveis

Carregue arquivos de ambiente, defina variáveis e interpole-as nas receitas. Mantenha a configuração organizada e fora dos seus comandos.

🌍 Multiplataforma

Um único binário estático que funciona da mesma forma no Linux, macOS e Windows — sem runtime, sem interpretador, sem esquisitices.

🏆 Benefícios de Produtividade

  • Economize Tempo: Execute qualquer tarefa do projeto com just <receita> em vez de lembrar comandos longos
  • Autodocumentado: just --list mostra cada tarefa e sua descrição
  • Consistente: Toda a equipe executa os mesmos comandos da mesma forma
  • Sem Esquisitices de Build: Receitas são comandos, não alvos — sem surpresas de timestamps desatualizados
  • Portátil: Um único justfile funciona no Linux, macOS e Windows

🆕 Novidades no just 1.56.0+

  • Novas Funções: Funções de expressão integradas adicionais para caminhos, strings e o ambiente
  • Atributos de Receita: Mais atributos para controlar como e quando as receitas são executadas
  • Melhor Autocompletar: Scripts de autocompletar aprimorados para bash, zsh e fish
  • Erros Mais Claros: Mensagens de erro de análise e de execução mais úteis
  • Multiplataforma: Melhorias contínuas para Windows e manipulação de shell
  • Página de Manual: Página de manual just.1 atualizada incluída no pacote

# Create a justfile in your project cat > justfile <<'EOF' # Build the project build: cargo build --release # Run the tests test: cargo test # Say hello to someone hello name='world': echo "Hello, {{name}}!" EOF # List available recipes just --list # Run a recipe just build just hello just

🎯 Exemplos de uso básico

Executando receitas:

# Run the first recipe in the justfile (the default) just # Run a specific recipe just test # Pass an argument to a recipe just hello Dario # List every recipe with its documentation just --list

Trabalhando com justfiles:

# Show the resolved value of a variable just --evaluate # Print a recipe without running it just --show build # Use a justfile in another directory just --justfile ~/project/justfile --working-directory ~/project build # Choose a recipe interactively (needs a chooser like fzf) just --choose

Autocompletar de shell:

# Completions for bash, fish and zsh are installed with the package. # Generate them manually for any shell if you need to: just --completions bash just --completions zsh just --completions fish

🚀 Por que escolher o deb.griffo.io?

📊 Comparação de Repositórios:
  • Debian Oficial: Disponível, mas frequentemente versões desatualizadas
  • Cargo Install: Requer toolchain Rust e compilação
  • Script de Instalação: Atualizações manuais e sem gerenciamento de pacotes
  • deb.griffo.io: Última versão (1.56.0+) com atualizações automáticas
  • ✅ Últimos Recursos: Obtenha os mais novos atributos e funções de receita
  • ✅ Sem Compilação Rust: Binários pré-compilados, sem necessidade de toolchain
  • ✅ Atualizações Automáticas: Pacotes atualizados em horas após os lançamentos upstream
  • ✅ Pacote Completo: Inclui autocompletar de shell e a página de manual
  • ✅ Multi-Distribuição: Funciona no Bookworm, Trixie e Sid
  • ✅ Manutenção Fácil: Comandos apt padrão para atualizações
