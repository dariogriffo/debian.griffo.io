O que é o Forgejo Runner?

O Forgejo Runner é o runner de CI para o Forgejo Actions. É um daemon leve que se conecta à sua instância Forgejo, pega jobs de workflow enfileirados e os executa na sua própria infraestrutura — usando Docker, LXC ou diretamente o host — proporcionando um pipeline de CI/CD totalmente auto-hospedado e compatível com o GitHub Actions.

🚀 Por que as últimas versões são importantes: O runner acompanha de perto os lançamentos do Forgejo, adicionando suporte a novos recursos de workflow, melhor isolamento e correções de bugs. Mantê-lo atualizado garante compatibilidade com os recursos mais recentes do Forgejo Actions.

⚡ Principais Recursos do Forgejo Runner

🔁 Compatível com Actions

Executa workflows usando a mesma sintaxe e o mesmo ecossistema de actions do GitHub Actions.

🐳 Execução Flexível

Execute jobs em containers Docker, LXC ou diretamente na máquina host.

🔐 Auto-hospedado

Mantenha seu CI/CD totalmente em sua própria infraestrutura, sem dependência de terceiros.

⚖️ Escalável

Registre múltiplos runners com labels para distribuir e direcionar cargas de trabalho.

🎯 Por que o Forgejo Runner?

  • Controle seu CI: Execute pipelines em hardware que você controla
  • Ecossistema de Actions: Reutilize definições de workflow no estilo GitHub existentes
  • Isolamento flexível: Execução via Docker, LXC ou host para atender às suas necessidades
  • Leve: Um único binário Go, simples de implantar

🆕 Destaques

  • Execução de Jobs: Executa workflows do Forgejo Actions enfileirados
  • Labels: Direcione jobs para runners específicos
  • Suporte a Containers: Builds isolados via Docker
  • Modo Daemon: Executa continuamente como um serviço

❓ Perguntas frequentes

O Forgejo Runner está nos repositórios oficiais do Debian?

Não — Forgejo Runner não está empacotado nos arquivos oficiais do Debian. Este repositório é atualmente a única fonte apt, oferecendo Forgejo Runner 12.13.1.

Como instalo o Forgejo Runner mais recente no Debian?

Adicione o repositório deb.griffo.io uma única vez com as instruções acima e execute: sudo apt install forgejo-runner. Novos lançamentos chegam pelo sudo apt upgrade normal.

Os pacotes são assinados e como são construídos?

Cada pacote é assinado com a chave GPG do repositório (EA0F721D231FDD3A0A17B9AC7808B4DD62C41256) e construído a partir de lançamentos upstream em repositórios públicos do GitHub que qualquer pessoa pode inspecionar.

Quais versões do Debian são suportadas?

Debian 12 Bookworm, Debian 13 Trixie, Forky e Sid.
