O que é o Neovim?

O Neovim é um editor de texto hiperextensível baseado no Vim que reestrutura a clássica base de código do Vim para o desenvolvimento moderno. Ele traz um cliente do Language Server Protocol integrado, sintaxe e compreensão de código com tecnologia Treesitter, um motor de tarefas assíncronas e scripting em Lua de primeira classe — para que plugins e configuração sejam rápidos, programáveis e fáceis de manter.

O arquivo oficial do Ubuntu traz o Neovim 0.11.6, e versões mais antigas ficam ainda mais para trás. Este repositório oferece o Neovim 0.12.4 igualmente no Ubuntu 22.04, 24.04, 25.10 e 26.04.

⚡ Principais Recursos do Neovim

🌙 Configuração em Lua

Configure e crie scripts para o editor em Lua, com uma API rica e estável. Uma linguagem de programação de verdade no lugar do Vimscript, para configurações rápidas e fáceis de manter.

🔌 LSP Integrado

Um cliente nativo do Language Server Protocol traz autocompletar, diagnósticos, ir para a definição e refatoração para qualquer linguagem que tenha um servidor.

🌳 Treesitter

Análise incremental para destaque de sintaxe, indentação e seleções estruturais precisos — muito além do destaque baseado em regex.

🚀 Assíncrono & Plugins

Um loop de eventos e tarefas não bloqueante mantém a interface responsiva, impulsionando um enorme ecossistema de plugins modernos em Lua e UIs remotas.

❓ Perguntas frequentes

O Neovim está nos repositórios oficiais do Ubuntu?

Sim, mas o arquivo oficial do Ubuntu traz Neovim 0.11.6, enquanto este repositório oferece Neovim 0.12.4, recompilado poucas horas após cada lançamento upstream.

Como instalo o Neovim mais recente no Ubuntu?

Adicione o repositório deb.griffo.io uma única vez com as instruções acima e execute: sudo apt install neovim-latest. Novos lançamentos chegam pelo sudo apt upgrade normal.

Os pacotes são assinados e como são construídos?

Cada pacote é assinado com a chave GPG do repositório (EA0F721D231FDD3A0A17B9AC7808B4DD62C41256) e construído a partir de lançamentos upstream em repositórios públicos do GitHub que qualquer pessoa pode inspecionar.
