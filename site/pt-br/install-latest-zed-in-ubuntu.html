O que é o Zed?

O Zed é um editor de código de nova geração projetado para colaboração de alto desempenho com humanos e IA. Criado pelos desenvolvedores do Atom e do Tree-sitter, o Zed foi escrito do zero em Rust para aproveitar eficientemente múltiplos núcleos de CPU e sua GPU, oferecendo uma experiência de edição incrivelmente rápida e responsiva.

🚀 Por que as versões mais recentes importam: O Zed está evoluindo rapidamente com novos recursos como suporte a depuração nativa, edição agêntica, integração com Git e melhorias de desempenho em cada lançamento. Mantenha-se atualizado para obter a melhor experiência de edição.

🔥 Principais Recursos do Zed

⚡ Extremamente Rápido

Escrito do zero em Rust para aproveitar eficientemente múltiplos núcleos de CPU e sua GPU. Experimente inicialização instantânea e rolagem suave a 120fps.

🤝 Edição Multiplayer

Colaboração em tempo real integrada. Converse com colegas de equipe, escreva notas juntos e compartilhe sua tela e projeto sem problemas.

🤖 Integração de IA

Integre LLMs ao seu fluxo de trabalho com edição agêntica, assistente inline e previsão de edição alimentada pelo modelo Zeta.

🐛 Depuração Nativa

Construído sobre o Protocolo de Adaptador de Depuração (DAP) com suporte nativo para depuração em múltiplas linguagens de programação.

🔧 Compatível com Vim

Edição modal de primeira classe por meio de atalhos do Vim, incluindo recursos como objetos de texto, marcadores e todos os seus movimentos favoritos.

🌐 Desenvolvimento Remoto

Sua máquina executa apenas a interface do Zed, enquanto a base de código real roda em um servidor remoto para um poderoso desenvolvimento em nuvem.
