⚡ Instalação rápida

Adicione este repositório ao seu sistema Debian e comece a instalar as versões mais recentes das suas ferramentas de desenvolvimento favoritas:

Passo 1: Adicionar repositório

curl -sS https://debian.griffo.io/EA0F721D231FDD3A0A17B9AC7808B4DD62C41256.asc | sudo gpg --dearmor --yes -o /etc/apt/trusted.gpg.d/debian.griffo.io.gpg echo "deb https://debian.griffo.io/apt $(lsb_release -sc 2>/dev/null) main" | sudo tee /etc/apt/sources.list.d/debian.griffo.io.list sudo apt update

🚀 Por que escolher debian.griffo.io?

Este repositório não oficial se especializa em fornecer as versões mais atualizadas de ferramentas de desenvolvimento populares como pacotes Debian corretamente empacotados. Enquanto os repositórios oficiais do Debian frequentemente ficam atrás dos lançamentos upstream, garantimos que você obtenha:

  • ✅ Versões estáveis mais recentes em horas após a publicação original
  • ✅ Builds noturnos para recursos de ponta (Zig, ZLS)
  • ✅ Empacotamento Debian adequado com dependências corretas
  • ✅ Suporte a múltiplas distribuições (Bookworm, Trixie, Forky, Sid)
  • ✅ Atualizações automatizadas garantindo disponibilidade contínua

⚠️ Informações importantes

Escopo: Este repositório se concentra exclusivamente em empacotamento não oficial para Debian. Para problemas com as ferramentas em si, entre em contato com os respectivos projetos originais.

Atualizações: Os pacotes são atualizados automaticamente quando novas versões são publicadas nos projetos originais, garantindo que você sempre tenha acesso às funcionalidades mais recentes e patches de segurança.

Comunidade: Este é um projeto impulsionado pela comunidade destinado a preencher a lacuna entre as ferramentas de desenvolvimento em rápida evolução e o ciclo de empacotamento estável do Debian.
