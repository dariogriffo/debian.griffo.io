⚡ Instalação rápida

Adicione este repositório ao seu sistema Debian e comece a instalar as versões mais recentes das suas ferramentas de desenvolvimento favoritas:

Passo 1: Adicionar repositório

sudo install -d -m 0755 /etc/apt/keyrings curl -fsSL https://deb.griffo.io/EA0F721D231FDD3A0A17B9AC7808B4DD62C41256.asc | sudo gpg --dearmor --yes -o /etc/apt/keyrings/deb.griffo.io.gpg echo "deb [signed-by=/etc/apt/keyrings/deb.griffo.io.gpg] https://deb.griffo.io/apt $(lsb_release -sc 2>/dev/null) main" | sudo tee /etc/apt/sources.list.d/deb.griffo.io.list > /dev/null sudo apt update

🚀 Por que escolher deb.griffo.io?

Este repositório não oficial se especializa em fornecer as versões mais atualizadas de ferramentas de desenvolvimento populares como pacotes Debian corretamente empacotados. Enquanto os repositórios oficiais do Debian frequentemente ficam atrás dos lançamentos upstream, garantimos que você obtenha:

  • ✅ Versões estáveis mais recentes em horas após a publicação original
  • ✅ Builds noturnos para recursos de ponta (Zig, ZLS)
  • ✅ Empacotamento Debian adequado com dependências corretas
  • ✅ Suporte a múltiplas distribuições (Bookworm, Trixie, Forky, Sid)
  • ✅ Atualizações automatizadas garantindo disponibilidade contínua
