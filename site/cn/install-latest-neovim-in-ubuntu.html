什么是 Neovim？

Neovim 是一款高度可扩展的、基于 Vim 的文本编辑器，它对经典的 Vim 代码库进行了重构，以适应现代开发。它内置了 Language Server Protocol 客户端、由 Treesitter 驱动的语法解析与代码理解、异步任务引擎，以及一等公民的 Lua 脚本支持——因此插件与配置既快速、可编程，又易于维护。

官方 Ubuntu 软件源提供 Neovim 0.11.6，更早的版本则落后更多。本仓库为 Ubuntu 22.04、24.04、25.10 和 26.04 提供 Neovim 0.12.4。

⚡ Neovim 的主要特性

🌙 Lua 配置

使用功能丰富且稳定的 API，以 Lua 配置并编写编辑器脚本。用真正的编程语言取代 Vimscript，实现快速、易维护的配置。

🔌 内置 LSP

原生的 Language Server Protocol 客户端，为任何拥有语言服务器的语言带来自动补全、诊断、跳转到定义和重构能力。

🌳 Treesitter

增量解析带来精确的语法高亮、缩进和结构化选择——远超基于正则表达式的高亮方式。

🚀 异步与插件

非阻塞的任务与事件循环让界面始终保持响应，驱动着庞大的现代 Lua 插件生态和远程 UI。

🚀 始终领先于官方软件源：Debian 和 Ubuntu 在发布时会冻结软件包版本——本仓库通常在上游发布后数小时内即发布新版本。只需 apt upgrade 即可始终保持最新版本。

❓ 常见问题

Neovim 在官方 Ubuntu 软件源中吗？

在，但官方 Ubuntu 源只提供 Neovim 0.11.6，而本仓库提供 Neovim 0.12.4，每次上游发布后数小时内重新构建。

如何在 Ubuntu 上安装最新版 Neovim？

按上方说明一次性添加 deb.griffo.io 仓库，然后运行：sudo apt install neovim-latest。之后通过常规的 sudo apt upgrade 获取新版本。

软件包是否有签名？如何构建？

每个软件包都使用仓库的 GPG 密钥（EA0F721D231FDD3A0A17B9AC7808B4DD62C41256）签名，并基于上游发布在公开的 GitHub 仓库中构建，任何人都可以审查。

支持哪些 Ubuntu 版本？

Ubuntu 22.04 Jammy、24.04 Noble、25.10 Questing 和 26.04 Resolute。
