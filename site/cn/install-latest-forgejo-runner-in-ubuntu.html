什么是 Forgejo Runner？

Forgejo Runner 是 Forgejo Actions 的CI 运行器。它是一个轻量级守护进程，连接到您的 Forgejo 实例，获取排队中的工作流任务，并在您自己的基础设施上执行——可使用 Docker、LXC 或直接在主机上运行——为您提供完全自托管、兼容 GitHub Actions 的 CI/CD 流水线。

🚀 为什么最新版本很重要：该运行器紧跟 Forgejo 的版本发布，持续加入对新工作流功能的支持、更好的隔离机制以及错误修复。保持最新可确保与最新的 Forgejo Actions 功能兼容。

⚡ Forgejo Runner 的核心功能

🔁 兼容 Actions

使用与 GitHub Actions 相同的语法和 action 生态系统运行工作流。

🐳 灵活执行

可在 Docker 容器、LXC 中执行任务，或直接在主机上运行。

🔐 自托管

将您的 CI/CD 完全保留在自己的基础设施上，无需依赖任何第三方。

⚖️ 可扩展

注册多个带标签的运行器，以分发和定向工作负载。

🎯 为什么选择 Forgejo Runner？

  • 掌控您的 CI：在您掌控的硬件上运行流水线
  • Actions 生态系统：复用现有的 GitHub 风格工作流定义
  • 灵活隔离：Docker、LXC 或主机执行，满足您的不同需求
  • 轻量级：单个 Go 二进制文件，部署简单

🆕 主要亮点

  • 任务执行：运行排队中的 Forgejo Actions 工作流
  • 标签：将任务定向到指定运行器
  • 容器支持：通过 Docker 实现隔离构建
  • 守护进程模式：作为服务持续运行
