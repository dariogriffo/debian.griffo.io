什么是 Oh My Posh？

Oh My Posh 是一个由 Jan De Dobbeleer 创建、使用 Go 编写的适用于任何 shell 的提示符主题引擎。一个引擎、一种主题格式——在 bash、zsh、fish、PowerShell、Nushell、elvish、xonsh 和 cmd 中呈现同样美观、信息丰富的提示符。凭借数万 GitHub star 和庞大的主题生态，它已成为在您使用的每台机器和每个 shell 上获得一致提示符的首选方式。

😤 提示符的痛点：还在手写 PS1 转义码？被只支持单一 shell 的提示符框架绑死？每次在家用 zsh、在服务器上用 bash 之间切换时，都要从头重建提示符？Oh My Posh 让一份配置跟随您到任何地方。
🚀 始终领先于官方软件源：Debian 和 Ubuntu 在发布时会冻结软件包版本——本仓库通常在上游发布后数小时内即发布新版本。只需 apt upgrade 即可始终保持最新版本。

⚡ Oh My Posh 的主要特性

🐚 任意 Shell，一份配置

在 bash、zsh、fish、PowerShell、Nushell、elvish、xonsh 和 cmd 中呈现完全一致的提示符。配置一次，处处可用——包括通过 SSH。

🎨 内置 125+ 主题

本软件包将所有内置主题安装到 /usr/share/oh-my-posh/themes/——可离线预览和使用，无需额外下载。

🧩 丰富的信息段

Git 状态、退出码、执行时间、电池、kubernetes 上下文、云账户，以及语言版本（Go、Python、Node、Rust 等数十种）——仅在相关时显示。

📝 您的主题，您的格式

按照文档化的 schema，使用 JSON、YAML 或 TOML 编写自定义主题。从任意内置主题出发，打造属于您自己的样式。

✨ 瞬态提示符与工具提示

瞬态提示符保持回滚区整洁，工具提示随您的输入实时响应，右对齐的信息段充分利用终端的全部宽度。

⚡ 天生快速

单个 Go 二进制文件，信息段并发渲染——即使启用大量信息段，提示符依然瞬时响应。

🏆 为什么用 apt 安装？

  • 无需 curl-pipe-bash：使用 GPG 签名的仓库，而不是把安装脚本直接通过管道送进您的 shell
  • 内置主题：所有内置主题安装到 /usr/share/oh-my-posh/themes/，与二进制文件版本匹配
  • 自动更新：新的上游版本随您日常的 apt upgrade 一同到达
  • 干净卸载：apt remove 一并移除所有内容——不会在 ~/bin 留下散落的二进制文件
  • 多架构支持：amd64、arm64 和 armhf——桌面和 Raspberry Pi 使用同一套安装方式

🚀 设置您的 shell

从 /usr/share/oh-my-posh/themes/ 中挑选一个主题并初始化您的 shell。对于 bash，将以下内容添加到 ~/.bashrc：

eval "$(oh-my-posh init bash --config /usr/share/oh-my-posh/themes/jandedobbeleer.omp.json)"

对于 zsh，将同一行（改用 zsh）添加到 ~/.zshrc；对于 fish，将 oh-my-posh init fish --config /usr/share/oh-my-posh/themes/jandedobbeleer.omp.json | source 添加到 ~/.config/fish/config.fish。大多数主题搭配 Nerd Font 效果最佳——也可以运行 oh-my-posh font install。
