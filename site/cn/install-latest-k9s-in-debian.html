什么是 k9s？

k9s 是一款用于管理 Kubernetes 集群的终端 UI 工具。它提供快速、键盘驱动的界面，让您能够实时观察和操作正在运行的集群——浏览资源、查看日志、进入 Pod 执行命令以及应用更改——而无需不断输入冗长的 kubectl 命令。

🚀 为什么最新版本很重要：k9s 频繁发布更新，带来新的视图、插件和 Kubernetes API 支持。保持版本最新可确保与最新集群版本和功能的兼容性。

⚡ k9s 的核心功能

⌨️ 键盘驱动

通过直观的快捷键，快速浏览和管理集群资源。

📜 实时日志与 Exec

直接在界面中流式查看 Pod 日志，并进入容器执行 Shell 命令。

🔍 资源视图

浏览 Deployment、Pod、Service 等资源，并实时查看其状态。

🧩 可扩展

自定义皮肤、别名和插件，打造专属于您的工作流。

🎯 为什么选择 k9s？

  • 比 kubectl 更快：无需输入冗长命令即可观察和操作资源
  • 实时性：实时查看集群状态和资源健康状况
  • 强大的导航：在相关资源之间即时跳转
  • 可定制：皮肤、快捷键和插件，适配您的团队

🆕 亮点功能

  • 集群导航：浏览所有 Kubernetes 资源类型
  • 日志与 Shell：追踪日志并进入 Pod 执行命令
  • 资源操作：在界面中扩缩容、删除和编辑资源
  • 插件：通过自定义命令进行扩展

❓ 常见问题

k9s 在官方 Debian 软件源中吗？

不在 — 官方 Debian 源没有打包 k9s。本仓库目前是唯一的 apt 来源，提供 k9s 0.51.0。

如何在 Debian 上安装最新版 k9s？

按上方说明一次性添加 deb.griffo.io 仓库，然后运行：sudo apt install k9s。之后通过常规的 sudo apt upgrade 获取新版本。

软件包是否有签名？如何构建？

每个软件包都使用仓库的 GPG 密钥（EA0F721D231FDD3A0A17B9AC7808B4DD62C41256）签名，并基于上游发布在公开的 GitHub 仓库中构建，任何人都可以审查。

支持哪些 Debian 版本？

Debian 12 Bookworm、Debian 13 Trixie、Forky 和 Sid。
