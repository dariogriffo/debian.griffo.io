什么是 Zed？

Zed 是一款下一代代码编辑器，专为与人类和 AI 的高性能协作而设计。由 Atom 和 Tree-sitter 的创建者打造，Zed 使用 Rust 从零开始编写，能够高效利用多个 CPU 核心和您的 GPU，提供极其快速和响应迅速的编辑体验。

🚀 为什么最新版本很重要：Zed 正在快速发展，每个版本都带来新功能，如原生调试支持、智能编辑、Git 集成和性能改进。保持更新以获得最佳编辑体验。

🔥 Zed 的主要特性

⚡ 极速运行

使用 Rust 从零开始编写，高效利用多个 CPU 核心和您的 GPU。体验即时启动和流畅的 120fps 滚动。

🤝 多人协作编辑

内置实时协作功能。与团队成员聊天、共同编写笔记，无缝共享您的屏幕和项目。

🤖 AI 集成

将大语言模型集成到您的工作流程中，包括智能编辑、内联助手和由 Zeta 模型驱动的编辑预测。

🐛 原生调试

基于调试适配器协议（DAP）构建，原生支持多种编程语言的调试。

🔧 Vim 友好

通过 Vim 键位绑定提供一流的模态编辑，包括文本对象、标记和所有您喜爱的移动命令等功能。

🌐 远程开发

您的机器只运行 Zed 用户界面，而实际代码库在远程服务器上运行，实现强大的云端开发。
