什么是 Starship？

Starship 是一款使用 Rust 编写的快速、可高度定制的跨 Shell 命令行提示符工具。它支持 Bash、Zsh、Fish、PowerShell 等多种 Shell，通过一个统一、一致且高度可配置的提示符，显示 git 状态、语言版本、云环境等丰富的上下文信息，即使在大型仓库中也能保持流畅响应。

🚀 为什么最新版本很重要：Starship 正在积极开发中，不断推出新模块、新集成和性能优化。运行最新版本可让您使用最新的提示符模块和配置选项。

⚡ Starship 的核心功能

⚡ 快速

使用 Rust 编写并经过优化，让您的提示符在任何环境下都保持响应迅速。

🐚 跨 Shell 支持

一个提示符适配 Bash、Zsh、Fish、PowerShell、Nushell 等多种 Shell。

🎨 完全可定制

通过单个 TOML 文件配置每个模块和符号。

🧩 上下文感知

仅在相关时显示 git 状态、语言版本、云环境等信息。

🎯 为什么选择 Starship？

  • 随处一致：在您所有的 Shell 和设备上使用同一个提示符
  • 丰富的上下文：一目了然地查看 Git、语言和环境信息
  • 高度可配置：通过 starship.toml 调整每一个细节
  • 快速：即使在大型仓库中延迟也极低

🆕 亮点特性

  • 跨 Shell：适配所有主流 Shell
  • 模块：内置数十个上下文模块
  • TOML 配置：单一、可移植的配置文件
  • 预设：Nerd Font 符号等现成的外观样式

❓ 常见问题

Starship 在官方 Debian 软件源中吗？

在，但官方 Debian 源只提供 Starship 1.22.1，而本仓库提供 Starship 1.26.0，每次上游发布后数小时内重新构建。

如何在 Debian 上安装最新版 Starship？

按上方说明一次性添加 deb.griffo.io 仓库，然后运行：sudo apt install starship。之后通过常规的 sudo apt upgrade 获取新版本。

软件包是否有签名？如何构建？

每个软件包都使用仓库的 GPG 密钥（EA0F721D231FDD3A0A17B9AC7808B4DD62C41256）签名，并基于上游发布在公开的 GitHub 仓库中构建，任何人都可以审查。

支持哪些 Debian 版本？

Debian 12 Bookworm、Debian 13 Trixie、Forky 和 Sid。
