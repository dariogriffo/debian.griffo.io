什么是 Headscale？

Headscale 是一款开源、自托管的 Tailscale 控制服务器实现。它让您可以运行自己的协调服务器，构建基于 WireGuard 的私有 mesh 网络，使您的设备之间可以安全地互联，而无需依赖 Tailscale 托管的控制平面——同时仍然使用标准的 Tailscale 客户端。

🚀 为什么最新版本很重要：Headscale 与 Tailscale 协议同步演进，不断加入对新客户端功能的支持、ACL 改进和稳定性修复。运行最新版本可以确保您与当前的 Tailscale 客户端保持兼容。

⚡ Headscale 的核心功能

🔐 自托管控制平面

运行您自己的协调服务器——无需依赖第三方托管服务。

🌐 WireGuard Mesh 网络

在您所有设备之间提供快速、加密的点对点连接。

📱 标准客户端

兼容桌面、移动设备和服务器上的官方 Tailscale 客户端。

📜 ACL 策略

细粒度的访问控制规则，用于划分和保护您的网络。

🎯 为什么选择 Headscale？

  • 完全掌控：拥有协调服务器和您的网络数据
  • 无厂商锁定：托管控制平面的开源替代方案
  • 使用官方客户端：保留精致的 Tailscale 应用体验
  • 轻量级：单个易于部署的 Go 二进制文件

🆕 亮点

  • 协调服务器：自托管的 Tailscale 控制平面
  • WireGuard：现代化的加密 mesh 网络
  • ACL：基于策略的访问控制
  • 多用户：使用命名空间分隔设备和用户
