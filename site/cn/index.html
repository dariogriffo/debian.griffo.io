🚀 为什么选择 debian.griffo.io？

这个非官方仓库专门提供最新版本的热门开发工具，作为正确打包的 Debian 软件包。虽然官方 Debian 仓库通常落后于上游发布，但我们确保您获得：

  • 最新稳定版本 - 上游发布后数小时内更新
  • 每夜构建 - 用于尖端功能（Zig、ZLS）
  • 正确的 Debian 打包 - 具有正确的依赖关系
  • 多发行版支持 - Bookworm、Trixie、Forky、Sid
  • 自动更新 - 确保持续可用性
  • 社区驱动 - 开源且透明

🐧 支持的 Debian 发行版

Debian Bookworm（旧稳定版） Debian Trixie（稳定版） Debian Forky（测试版） Debian Sid（不稳定版）

这是一个非官方社区项目，提供官方 Debian 仓库中不可用的软件包。

范围：本仓库专注于非官方 Debian 打包。有关工具本身的问题，请联系相应的上游项目。

更新：当上游发布新版本时，软件包会自动更新，确保您始终可以访问最新功能和安全补丁。

社区：这是一个社区驱动的项目，旨在弥合快速发展的开发工具和 Debian 稳定打包周期之间的差距。
