¿Qué es Neovim?

Neovim es un editor de texto hiperextensible basado en Vim que refactoriza el clásico código de Vim para el desarrollo moderno. Incluye un cliente integrado del Language Server Protocol, análisis de sintaxis y comprensión del código gracias a Treesitter, un motor de tareas asíncrono y scripting en Lua de primera clase — de modo que los plugins y la configuración son rápidos, programables y fáciles de mantener.

El Debian oficial trixie trae Neovim 0.10.4, y Bookworm se queda mucho más atrás. Este repositorio ofrece Neovim 0.12.4 por igual en Bookworm, Trixie, Forky y Sid.

⚡ Características principales de Neovim

🌙 Configuración en Lua

Configura y programa el editor en Lua con una API rica y estable. Un lenguaje de programación de verdad en lugar de Vimscript para configuraciones rápidas y fáciles de mantener.

🔌 LSP integrado

Un cliente nativo del Language Server Protocol aporta autocompletado, diagnósticos, ir a la definición y refactorización a cualquier lenguaje que tenga un servidor.

🌳 Treesitter

Análisis incremental para un resaltado de sintaxis preciso, indentación y selecciones estructurales — mucho más allá del resaltado basado en expresiones regulares.

🚀 Asincronía y plugins

Un bucle de eventos y tareas no bloqueante mantiene la interfaz fluida, impulsando un enorme ecosistema de plugins modernos en Lua e interfaces remotas.
