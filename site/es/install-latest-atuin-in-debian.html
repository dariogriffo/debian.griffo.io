¿Qué es Atuin?

Atuin reemplaza tu historial de shell con una base de datos SQLite buscable, añadiendo una potente búsqueda de texto completo, sincronización cifrada de extremo a extremo opcional entre máquinas y estadísticas útiles. Registra contexto adicional para cada comando — código de salida, duración, directorio y más — y funciona con Bash, Zsh, Fish y otras shells.

🚀 Por qué importan las últimas versiones: Atuin se desarrolla activamente con mejoras en la búsqueda, la sincronización y la interfaz interactiva. Ejecutar la última versión te da acceso a las funciones y correcciones más recientes del historial.

⚡ Características clave de Atuin

🔍 Búsqueda potente

Búsqueda de texto completo, difusa y por prefijo en todo tu historial de shell.

🔐 Sincronización cifrada

Sincronización cifrada de extremo a extremo opcional del historial entre todas tus máquinas.

📊 Contexto enriquecido

Registra el código de salida, la duración, el directorio y más para cada comando.

🐚 Multi-shell

Funciona con Bash, Zsh, Fish y otras shells populares.

🎯 ¿Por qué Atuin?

  • Mejor historial: Historial SQLite buscable que nunca se trunca silenciosamente
  • Sincronización entre máquinas: Sincronización cifrada de extremo a extremo y autoalojable
  • Rico en contexto: Sabe cuándo, dónde y cómo se ejecutó cada comando
  • Estadísticas: Información sobre tus comandos más usados

🆕 Novedades

  • Historial SQLite: Historial de comandos duradero y consultable
  • Búsqueda interactiva: Búsqueda difusa vinculada a la flecha hacia arriba
  • Sincronización: Sincronización cifrada opcional entre varias máquinas
  • Estadísticas: Estadísticas de uso de comandos
