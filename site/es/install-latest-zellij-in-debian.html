¿Qué es Zellij?

Zellij es un espacio de trabajo y multiplexor de terminal escrito en Rust. Una alternativa moderna a tmux y screen, te permite dividir tu terminal en paneles y pestañas, guardar y restaurar diseños, y extenderlo con un sistema de plugins WebAssembly — todo con combinaciones de teclas fáciles de descubrir y una interfaz amigable que lo hacen sencillo de aprender.

🚀 Por qué importan las últimas versiones: Zellij se desarrolla activamente con nuevas funciones de diseño, capacidades de plugins y mejoras de rendimiento. Ejecutar la última versión te da las funciones y correcciones más recientes del espacio de trabajo.

⚡ Características clave de Zellij

🪟 Paneles y pestañas

Divide tu terminal en paneles y organiza el trabajo en varias pestañas.

📐 Diseños

Define y restaura diseños de espacio de trabajo reproducibles para tus proyectos.

🔌 Sistema de plugins

Extiende Zellij con plugins WebAssembly para funcionalidades personalizadas.

🧭 Interfaz fácil de descubrir

Las indicaciones en pantalla hacen que las combinaciones de teclas sean fáciles de aprender y recordar.

🎯 ¿Por qué Zellij?

  • Fácil para principiantes: Las indicaciones de teclas en pantalla reducen la curva de aprendizaje
  • Diseños potentes: Espacios de trabajo reproducibles definidos en la configuración
  • Extensible: Sistema de plugins WebAssembly
  • Moderno: Implementación rápida en Rust con valores predeterminados sensatos

🆕 Novedades

  • Multiplexación: Paneles y pestañas en una sola terminal
  • Gestión de sesiones: Desconéctate y vuelve a conectar a sesiones en ejecución
  • Diseños: Guarda y carga configuraciones del espacio de trabajo
  • Plugins: Extensibilidad basada en WASM

❓ Preguntas frecuentes

¿Está Zellij en los repositorios oficiales de Debian?

No — Zellij no está empaquetado en los archivos oficiales de Debian. Este repositorio es actualmente la única fuente apt y ofrece Zellij 0.44.3.

¿Cómo instalo el último Zellij en Debian?

Añade el repositorio deb.griffo.io una sola vez con las instrucciones de arriba y ejecuta: sudo apt install zellij. Las nuevas versiones llegan con el sudo apt upgrade habitual.

¿Los paquetes están firmados y cómo se compilan?

Cada paquete está firmado con la clave GPG del repositorio (EA0F721D231FDD3A0A17B9AC7808B4DD62C41256) y se compila a partir de versiones upstream en repositorios públicos de GitHub que cualquiera puede inspeccionar.

Debian 12 Bookworm, Debian 13 Trixie, Forky y Sid.
