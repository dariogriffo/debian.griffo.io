⚡ Instalación rápida

Agregue este repositorio a su sistema Debian y comience a instalar las últimas versiones de sus herramientas de desarrollo favoritas:

Paso 1: Agregar repositorio

sudo install -d -m 0755 /etc/apt/keyrings curl -fsSL https://deb.griffo.io/EA0F721D231FDD3A0A17B9AC7808B4DD62C41256.asc | sudo gpg --dearmor --yes -o /etc/apt/keyrings/deb.griffo.io.gpg echo "deb [signed-by=/etc/apt/keyrings/deb.griffo.io.gpg] https://deb.griffo.io/apt $(lsb_release -sc 2>/dev/null) main" | sudo tee /etc/apt/sources.list.d/deb.griffo.io.list > /dev/null sudo apt update

🚀 ¿Por qué elegir deb.griffo.io?

Este repositorio no oficial se especializa en proporcionar las versiones más actualizadas de herramientas de desarrollo populares como paquetes de Debian correctamente empaquetados. Mientras que los repositorios oficiales de Debian a menudo se quedan atrás de los lanzamientos upstream, nos aseguramos de que obtenga:

  • ✅ Últimas versiones estables en horas después de la publicación original
  • ✅ Compilaciones nocturnas para características de vanguardia (Zig, ZLS)
  • ✅ Empaquetado Debian adecuado con dependencias correctas
  • ✅ Soporte para múltiples distribuciones (Bookworm, Trixie, Forky, Sid)
  • ✅ Actualizaciones automatizadas garantizando disponibilidad continua
