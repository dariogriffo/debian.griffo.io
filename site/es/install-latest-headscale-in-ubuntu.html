¿Qué es Headscale?

Headscale es una implementación de código abierto y autoalojada del servidor de control de Tailscale. Te permite ejecutar tu propio servidor de coordinación para una red mesh privada basada en WireGuard, de modo que tus dispositivos puedan conectarse entre sí de forma segura sin depender del plano de control alojado de Tailscale — mientras sigues usando los clientes estándar de Tailscale.

🚀 Por qué importan las últimas versiones: Headscale evoluciona junto con el protocolo de Tailscale, añadiendo soporte para nuevas funciones de cliente, mejoras en las ACL y correcciones de estabilidad. Ejecutar la última versión te mantiene compatible con los clientes actuales de Tailscale.

⚡ Características clave de Headscale

🔐 Plano de control autoalojado

Ejecuta tu propio servidor de coordinación — sin depender de un servicio alojado por terceros.

🌐 Malla WireGuard

Conectividad punto a punto rápida y cifrada entre todos tus dispositivos.

📱 Clientes estándar

Funciona con los clientes oficiales de Tailscale en escritorio, móvil y servidores.

📜 Políticas ACL

Reglas de control de acceso granular para segmentar y proteger tu red.

🎯 ¿Por qué Headscale?

  • Control total: Eres dueño del servidor de coordinación y de los datos de tu red
  • Sin dependencia de proveedor: Alternativa de código abierto al plano de control alojado
  • Usa clientes oficiales: Conserva la experiencia pulida de la app de Tailscale
  • Ligero: Un único binario de Go fácil de desplegar

🆕 Aspectos destacados

  • Servidor de coordinación: Plano de control de Tailscale autoalojado
  • WireGuard: Redes mesh modernas y cifradas
  • ACL: Control de acceso basado en políticas
  • Multiusuario: Espacios de nombres para separar dispositivos y usuarios
