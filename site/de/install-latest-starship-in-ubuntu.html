Was ist Starship?

Starship ist ein schneller, anpassbarer Cross-Shell-Prompt, geschrieben in Rust. Er funktioniert mit Bash, Zsh, Fish, PowerShell und weiteren Shells und zeigt kontextbezogene Informationen — Git-Status, Sprachversionen, Cloud-Kontext und vieles mehr — über einen einzigen, konsistenten und stark konfigurierbaren Prompt, der auch in großen Repositories flott bleibt.

🚀 Warum neueste Versionen wichtig sind: Starship wird aktiv weiterentwickelt, mit neuen Modulen, Integrationen und Performance-Optimierungen. Mit dem neuesten Release erhalten Sie die aktuellsten Prompt-Module und Konfigurationsoptionen.

⚡ Wichtigste Funktionen von Starship

⚡ Schnell

In Rust geschrieben und darauf optimiert, Ihren Prompt überall reaktionsschnell zu halten.

🐚 Cross-Shell

Ein Prompt für Bash, Zsh, Fish, PowerShell, Nushell und weitere.

🎨 Vollständig anpassbar

Konfigurieren Sie jedes Modul und Symbol über eine einzige TOML-Datei.

🧩 Kontextbewusst

Zeigt Git-Status, Sprachversionen, Cloud-Kontext und mehr, nur wenn relevant.

🎯 Warum Starship?

  • Überall konsistent: Derselbe Prompt in allen Ihren Shells und auf allen Rechnern
  • Reichhaltiger Kontext: Git, Sprachen und Umgebung auf einen Blick
  • Stark konfigurierbar: Passen Sie jedes Detail über starship.toml an
  • Schnell: Minimale Latenz auch in großen Repositories

🆕 Highlights

  • Cross-Shell: Funktioniert in jeder gängigen Shell
  • Module: Dutzende integrierte Kontextmodule
  • TOML-Konfiguration: Eine einzige, portable Konfigurationsdatei
  • Presets: Vorgefertigte Looks wie Nerd-Font-Symbole
