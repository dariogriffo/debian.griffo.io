⚡ Schnellinstallation

Fügen Sie dieses Repository zu Ihrem Debian-System hinzu und beginnen Sie mit der Installation der neuesten Versionen Ihrer bevorzugten Entwicklungstools:

Schritt 1: Repository hinzufügen

sudo install -d -m 0755 /etc/apt/keyrings curl -fsSL https://deb.griffo.io/EA0F721D231FDD3A0A17B9AC7808B4DD62C41256.asc | sudo gpg --dearmor --yes -o /etc/apt/keyrings/deb.griffo.io.gpg echo "deb [signed-by=/etc/apt/keyrings/deb.griffo.io.gpg] https://deb.griffo.io/apt $(lsb_release -sc 2>/dev/null) main" | sudo tee /etc/apt/sources.list.d/deb.griffo.io.list > /dev/null sudo apt update

Pakete nach dem Hinzufügen des Repositorys suchen

Sobald das Repository hinzugefügt ist, können Sie Standard-apt-Werkzeuge verwenden, um verfügbare Pakete zu erkunden:

# Alle Pakete dieses Repositorys auflisten apt-cache search --names-only '' | grep -v '^lib' # Details zu einem bestimmten Paket anzeigen apt-cache show zig # Prüfen, welche Version installiert wird und aus welcher Quelle apt-cache policy zig

🚀 Warum deb.griffo.io wählen?

Dieses inoffizielle Repository ist darauf spezialisiert, die aktuellsten Versionen beliebter Entwicklungstools als ordnungsgemäß gepackte Debian-Pakete bereitzustellen. Während offizielle Debian-Repositories oft hinter Upstream-Releases zurückbleiben, stellen wir sicher, dass Sie erhalten:

  • ✅ Latest stable releases within hours of upstream publication
  • ✅ Nightly builds for cutting-edge features (Zig, ZLS)
  • ✅ Proper Debian packaging with correct dependencies
  • ✅ Multiple distribution support (Bookworm, Trixie, Forky, Sid)
  • ✅ Automated updates ensuring continuous availability
