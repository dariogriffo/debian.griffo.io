Was ist Forgejo Runner?

Forgejo Runner ist der CI-Runner für Forgejo Actions. Es handelt sich um einen schlanken Daemon, der sich mit Ihrer Forgejo-Instanz verbindet, wartende Workflow-Jobs entgegennimmt und diese auf Ihrer eigenen Infrastruktur ausführt — über Docker, LXC oder direkt auf dem Host —, sodass Sie eine vollständig selbst gehostete, zu GitHub Actions kompatible CI/CD-Pipeline erhalten.

🚀 Warum neueste Versionen wichtig sind: Der Runner folgt den Forgejo-Releases eng und bringt Unterstützung für neue Workflow-Funktionen, bessere Isolation und Fehlerbehebungen mit sich. Wer ihn aktuell hält, stellt die Kompatibilität mit den neuesten Forgejo-Actions-Funktionen sicher.

⚡ Wichtigste Funktionen von Forgejo Runner

🔁 Actions-kompatibel

Führt Workflows mit derselben Syntax und demselben Actions-Ökosystem wie GitHub Actions aus.

🐳 Flexible Ausführung

Jobs in Docker-Containern, LXC oder direkt auf der Host-Maschine ausführen.

🔐 Selbst gehostet

Behalten Sie Ihre CI/CD vollständig auf Ihrer eigenen Infrastruktur — ganz ohne Abhängigkeit von Drittanbietern.

⚖️ Skalierbar

Registrieren Sie mehrere Runner mit Labels, um Workloads zu verteilen und gezielt anzusprechen.

🎯 Warum Forgejo Runner?

  • Eigene CI: Pipelines auf Hardware ausführen, die Sie selbst kontrollieren
  • Actions-Ökosystem: Bestehende Workflow-Definitionen im GitHub-Stil weiterverwenden
  • Flexible Isolation: Docker-, LXC- oder Host-Ausführung je nach Bedarf
  • Leichtgewichtig: Eine einzige Go-Binärdatei, die sich einfach bereitstellen lässt

🆕 Highlights

  • Job-Ausführung: Führt wartende Forgejo-Actions-Workflows aus
  • Labels: Jobs gezielt an bestimmte Runner weiterleiten
  • Container-Unterstützung: Isolierte Builds via Docker
  • Daemon-Modus: Läuft dauerhaft als Dienst

❓ Häufig gestellte Fragen

Ist Forgejo Runner in den offiziellen Ubuntu-Paketquellen?

Nein — Forgejo Runner ist nicht in den offiziellen Ubuntu-Archiven paketiert. Dieses Repository ist derzeit die einzige apt-Quelle und liefert Forgejo Runner 12.13.1.

Wie installiere ich das neueste Forgejo Runner unter Ubuntu?

Fügen Sie das deb.griffo.io-Repository einmalig hinzu (Anleitung oben) und führen Sie dann aus: sudo apt install forgejo-runner. Neue Releases kommen über das normale sudo apt upgrade.

Sind die Pakete signiert und wie werden sie gebaut?

Jedes Paket ist mit dem GPG-Schlüssel des Repositories (EA0F721D231FDD3A0A17B9AC7808B4DD62C41256) signiert und wird aus Upstream-Releases in öffentlichen GitHub-Repositories gebaut, die jeder einsehen kann.

Welche Ubuntu-Versionen werden unterstützt?

Ubuntu 22.04 Jammy, 24.04 Noble, 25.10 Questing und 26.04 Resolute.
