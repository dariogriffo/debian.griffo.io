Was ist Oh My Posh?

Oh My Posh ist eine Prompt-Theme-Engine für jede Shell, entwickelt von Jan De Dobbeleer und geschrieben in Go. Eine Engine, ein Theme-Format — derselbe schöne, informationsreiche Prompt in bash, zsh, fish, PowerShell, Nushell, elvish, xonsh und cmd. Mit Zehntausenden GitHub-Sternen und einem riesigen Theme-Ökosystem ist es der bevorzugte Weg zu einem konsistenten Prompt auf jeder Maschine und in jeder Shell, die Sie verwenden.

😤 Typische Prompt-Probleme: Sie basteln PS1-Escape-Codes von Hand? Sie sind an ein Prompt-Framework gebunden, das nur in einer Shell funktioniert? Sie bauen Ihren Prompt jedes Mal neu, wenn Sie zwischen zsh zu Hause und bash auf Servern wechseln? Oh My Posh gibt Ihnen eine Konfiguration, die Ihnen überallhin folgt.
🚀 Immer aktueller als die offiziellen Archive: Debian und Ubuntu frieren Paketversionen mit jedem Release ein — dieses Repository veröffentlicht neue Upstream-Versionen in der Regel innerhalb weniger Stunden. Ein einfaches apt upgrade hält Sie immer auf der neuesten Version.

⚡ Hauptfunktionen von Oh My Posh

🐚 Jede Shell, eine Konfiguration

Identischer Prompt in bash, zsh, fish, PowerShell, Nushell, elvish, xonsh und cmd. Einmal konfigurieren, überall nutzen — auch über SSH.

🎨 125+ Themes enthalten

Dieses Paket liefert jedes integrierte Theme nach /usr/share/oh-my-posh/themes/ — bereit zur Vorschau und offline nutzbar, ohne zusätzliche Downloads.

🧩 Umfangreiche Segmente

Git-Status, Exit-Codes, Ausführungszeit, Akku, Kubernetes-Kontext, Cloud-Konten und Sprachversionen (Go, Python, Node, Rust und Dutzende mehr) — nur angezeigt, wenn relevant.

📝 Ihr Theme, Ihr Format

Schreiben Sie eigene Themes in JSON, YAML oder TOML mit einem dokumentierten Schema. Starten Sie mit einem beliebigen integrierten Theme und machen Sie es zu Ihrem eigenen.

✨ Transient & Tooltips

Transiente Prompts halten den Verlauf sauber, Tooltips reagieren auf Ihre Eingabe, und rechtsbündige Segmente nutzen die volle Terminalbreite.

⚡ Schnell durch Design

Ein einzelnes Go-Binary mit nebenläufigem Segment-Rendering — der Prompt bleibt auch mit vielen aktivierten Segmenten verzögerungsfrei.

🏆 Warum mit apt installieren?

  • Kein curl-pipe-bash: ein GPG-signiertes Repository statt Installer-Skripte in Ihre Shell zu leiten
  • Themes enthalten: alle integrierten Themes landen in /usr/share/oh-my-posh/themes/, versionsgleich mit dem Binary
  • Automatische Updates: neue Upstream-Versionen kommen mit Ihrem normalen apt upgrade
  • Saubere Deinstallation: apt remove entfernt alles restlos — keine verwaisten Binaries in ~/bin
  • Multi-Arch: amd64, arm64 und armhf — dasselbe Setup auf Ihrem Desktop und Ihrem Raspberry Pi

🚀 Shell einrichten

Wählen Sie ein Theme aus /usr/share/oh-my-posh/themes/ und initialisieren Sie Ihre Shell. Für bash fügen Sie dies zu ~/.bashrc hinzu:

eval "$(oh-my-posh init bash --config /usr/share/oh-my-posh/themes/jandedobbeleer.omp.json)"

Für zsh fügen Sie dieselbe Zeile mit zsh zu ~/.zshrc hinzu; für fish fügen Sie oh-my-posh init fish --config /usr/share/oh-my-posh/themes/jandedobbeleer.omp.json | source zu ~/.config/fish/config.fish hinzu. Die meisten Themes sehen mit einer Nerd Font am besten aus — oder führen Sie oh-my-posh font install aus.

❓ Häufig gestellte Fragen

Ist Oh My Posh in den offiziellen Ubuntu-Paketquellen?

Nein — Oh My Posh ist nicht in den offiziellen Ubuntu-Archiven paketiert. Dieses Repository ist derzeit die einzige apt-Quelle und liefert Oh My Posh 29.35.3.

Wie installiere ich das neueste Oh My Posh unter Ubuntu?

Fügen Sie das deb.griffo.io-Repository einmalig hinzu (Anleitung oben) und führen Sie dann aus: sudo apt install oh-my-posh. Neue Releases kommen über das normale sudo apt upgrade.

Sind die Pakete signiert und wie werden sie gebaut?

Jedes Paket ist mit dem GPG-Schlüssel des Repositories (EA0F721D231FDD3A0A17B9AC7808B4DD62C41256) signiert und wird aus Upstream-Releases in öffentlichen GitHub-Repositories gebaut, die jeder einsehen kann.

Welche Ubuntu-Versionen werden unterstützt?

Ubuntu 22.04 Jammy, 24.04 Noble, 25.10 Questing und 26.04 Resolute.
