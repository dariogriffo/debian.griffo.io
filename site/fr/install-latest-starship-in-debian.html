Qu'est-ce que Starship ?

Starship est une invite de commande rapide et personnalisable, compatible avec tous les shells, écrite en Rust. Elle fonctionne avec Bash, Zsh, Fish, PowerShell et bien d'autres, en affichant des informations contextuelles — état git, versions des langages, contexte cloud et bien plus encore — via une invite unique, cohérente et hautement configurable qui reste réactive même dans les gros dépôts.

🚀 Pourquoi les dernières versions sont importantes : Starship est activement développé, avec de nouveaux modules, des intégrations et des optimisations de performance. Utiliser la dernière version vous donne accès aux modules d'invite les plus récents et à de nouvelles options de configuration.

⚡ Fonctionnalités clés de Starship

⚡ Rapide

Écrit en Rust et optimisé pour garder votre invite réactive en toutes circonstances.

🐚 Compatible avec tous les shells

Une seule invite pour Bash, Zsh, Fish, PowerShell, Nushell et bien d'autres.

🎨 Entièrement personnalisable

Configurez chaque module et symbole via un unique fichier TOML.

🧩 Sensible au contexte

Affiche l'état git, les versions des langages, le contexte cloud et bien plus, uniquement lorsque c'est pertinent.

🎯 Pourquoi Starship ?

  • Cohérent partout : La même invite sur tous vos shells et toutes vos machines
  • Contexte riche : Git, langages et environnement en un coup d'œil
  • Hautement configurable : Ajustez chaque détail via starship.toml
  • Rapide : Latence minimale, même dans les gros dépôts

🆕 Points forts

  • Compatible avec tous les shells : Fonctionne dans tous les shells les plus courants
  • Modules : Des dizaines de modules contextuels intégrés
  • Configuration TOML : Un seul fichier de configuration portable
  • Préréglages : Des styles prêts à l'emploi, comme les symboles Nerd Font

🎯 Exemples d'utilisation de base

# Show version starship --version # Add to ~/.bashrc echo 'eval "$(starship init bash)"' >> ~/.bashrc # Add to ~/.zshrc echo 'eval "$(starship init zsh)"' >> ~/.zshrc # Add to Fish (~/.config/fish/config.fish) echo 'starship init fish | source' >> ~/.config/fish/config.fish # Print the effective configuration starship print-config
