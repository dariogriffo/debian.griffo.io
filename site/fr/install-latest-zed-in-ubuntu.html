Qu'est-ce que Zed ?

Zed est un éditeur de code de nouvelle génération conçu pour une collaboration haute performance avec les humains et l'IA. Créé par les développeurs d'Atom et de Tree-sitter, Zed est écrit de zéro en Rust pour exploiter efficacement plusieurs cœurs de processeur et votre GPU, offrant une expérience d'édition incroyablement rapide et réactive.

🚀 Pourquoi les dernières versions sont importantes : Zed évolue rapidement avec de nouvelles fonctionnalités comme le support de débogage natif, l'édition agentique, l'intégration Git et des améliorations de performance à chaque version. Restez à jour pour obtenir la meilleure expérience d'édition.

🔥 Fonctionnalités principales de Zed

⚡ Extrêmement rapide

Écrit de zéro en Rust pour exploiter efficacement plusieurs cœurs de processeur et votre GPU. Profitez d'un démarrage instantané et d'un défilement fluide à 120fps.

🤝 Édition multijoueur

Collaboration en temps réel intégrée. Discutez avec vos coéquipiers, écrivez des notes ensemble et partagez votre écran et votre projet de manière transparente.

🤖 Intégration IA

Intégrez des LLM dans votre flux de travail avec l'édition agentique, l'assistant en ligne et la prédiction d'édition alimentée par le modèle Zeta.

🐛 Débogage natif

Construit sur le protocole d'adaptateur de débogage (DAP) avec support natif pour le débogage dans plusieurs langages de programmation.

🔧 Compatible Vim

Édition modale de première classe via les raccourcis Vim, incluant des fonctionnalités comme les objets texte, les marques et tous vos mouvements préférés.

🌐 Développement à distance

Votre machine exécute uniquement l'interface Zed, tandis que le code réel s'exécute sur un serveur distant pour un développement cloud puissant.
