deb.griffo.io

Dépôt non officiel pour les derniers outils de développement

🚀 Toujours à jour : Ce dépôt fournit les versions les plus récentes d'outils de développement populaires sous forme de paquets Debian, mis à jour régulièrement pour garantir que vous avez accès aux dernières fonctionnalités et correctifs de sécurité.

⚡ Installation rapide

Ajoutez ce dépôt à votre système Debian et commencez à installerer les dernières versions de vos outils de développement préférés :

Étape 1 : Ajouter le dépôt

sudo install -d -m 0755 /etc/apt/keyrings curl -fsSL https://deb.griffo.io/EA0F721D231FDD3A0A17B9AC7808B4DD62C41256.asc | sudo gpg --dearmor --yes -o /etc/apt/keyrings/deb.griffo.io.gpg echo "deb [signed-by=/etc/apt/keyrings/deb.griffo.io.gpg] https://deb.griffo.io/apt $(lsb_release -sc 2>/dev/null) main" | sudo tee /etc/apt/sources.list.d/deb.griffo.io.list > /dev/null sudo apt update

🔍 Parcourir le dépôt de paquets

Vous pouvez parcourir le dépôt de paquets directement sur https://deb.griffo.io/apt/dists/. Cela vous permet d'explorer les distributions disponibles, les composants et les fichiers de paquets individuels sans avoir à ajouter le dépôt au préalable.

Structure des répertoires

Le dépôt suit la disposition standard des dépôts Debian :

https://deb.griffo.io/apt/dists/ ← index des distributions https://deb.griffo.io/apt/dists/bookworm/ ← Debian Bookworm (12) https://deb.griffo.io/apt/dists/trixie/ ← Debian Trixie (13) https://deb.griffo.io/apt/dists/forky/ ← Debian Forky (testing) https://deb.griffo.io/apt/dists/sid/ ← Debian Sid (unstable) https://deb.griffo.io/apt/dists/jammy/ ← Ubuntu Jammy (22.04 LTS) https://deb.griffo.io/apt/dists/noble/ ← Ubuntu Noble (24.04 LTS)

Rechercher des paquets après l'ajout du dépôt

Une fois le dépôt ajouté, vous pouvez utiliser les outils apt standard pour explorer les paquets disponibles :

# Lister tous les paquets fournis par ce dépôt apt-cache search --names-only '' | grep -v '^lib' # Afficher les détails d'un paquet spécifique apt-cache show zig # Vérifier quelle version sera installée et depuis quelle source apt-cache policy zig

🚀 Pourquoi choisir deb.griffo.io ?

Ce dépôt non officiel se spécialise dans la fourniture des versions les plus récentes d'outils de développement populaires sous forme de paquets Debian correctement empaquetés. Alors que les dépôts Debian officiels sont souvent en retard par rapport aux versions upstream, nous nous assurons que vous obtenez :

  • ✅ Latest stable releases within hours of upstream publication
  • ✅ Nightly builds for cutting-edge features (Zig, ZLS)
  • ✅ Proper Debian packaging with correct dependencies
  • ✅ Multiple distribution support (Bookworm, Trixie, Forky, Sid)
  • ✅ Automated updates ensuring continuous availability

Scope: This repository focuses exclusively on unofficial Debian packaging. For issues with the tools themselves, please contact the respective upstream projects.

Community: This is a community-driven project aimed at bridging the gap between fast-moving outils de développement and Debian's stable packaging cycle.
