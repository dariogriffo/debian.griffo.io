Qu'est-ce qu'Oh My Posh ?

Oh My Posh est un moteur de thèmes de prompt pour tous les shells, créé par Jan De Dobbeleer et écrit en Go. Un seul moteur, un seul format de thème — le même prompt élégant et riche en informations dans bash, zsh, fish, PowerShell, Nushell, elvish, xonsh et cmd. Avec des dizaines de milliers d'étoiles sur GitHub et un immense écosystème de thèmes, il est devenu la référence pour obtenir un prompt cohérent sur toutes les machines et tous les shells que vous utilisez.

😤 Les galères du prompt : Vous écrivez vos codes d'échappement PS1 à la main ? Prisonnier d'un framework de prompt qui ne fonctionne que dans un seul shell ? Vous reconstruisez votre prompt de zéro chaque fois que vous passez de zsh à la maison à bash sur les serveurs ? Oh My Posh vous offre une configuration unique qui vous suit partout.
🚀 Toujours en avance sur les archives officielles : Debian et Ubuntu figent les versions des paquets à chaque publication — ce dépôt publie les nouvelles versions upstream généralement en quelques heures. Un simple apt upgrade vous garde toujours à la dernière version.

⚡ Fonctionnalités clés d'Oh My Posh

🐚 Tous les shells, une seule config

Un prompt identique dans bash, zsh, fish, PowerShell, Nushell, elvish, xonsh et cmd. Configurez une fois, utilisez partout — y compris via SSH.

🎨 125+ thèmes inclus

Ce paquet installe tous les thèmes intégrés dans /usr/share/oh-my-posh/themes/ — prêts à être prévisualisés et utilisés hors ligne, sans téléchargement supplémentaire.

🧩 Segments riches

Statut Git, codes de sortie, temps d'exécution, batterie, contexte kubernetes, comptes cloud et versions de langages (Go, Python, Node, Rust et des dizaines d'autres) — affichés uniquement quand c'est pertinent.

📝 Votre thème, votre format

Écrivez des thèmes personnalisés en JSON, YAML ou TOML avec un schéma documenté. Partez de n'importe quel thème intégré et faites-en le vôtre.

✨ Transient & Tooltips

Les prompts transients gardent l'historique du terminal propre, les tooltips réagissent à ce que vous tapez, et les segments alignés à droite exploitent toute la largeur du terminal.

⚡ Rapide par conception

Un unique binaire Go avec rendu concurrent des segments — le prompt reste instantané même avec de nombreux segments activés.

🏆 Pourquoi l'installer avec apt ?

  • Pas de curl-pipe-bash : un dépôt signé GPG au lieu de scripts d'installation envoyés directement dans votre shell
  • Thèmes inclus : tous les thèmes intégrés arrivent dans /usr/share/oh-my-posh/themes/, avec la version correspondant au binaire
  • Mises à jour automatiques : les nouvelles versions upstream arrivent avec votre apt upgrade habituel
  • Désinstallation propre : apt remove emporte tout avec lui — pas de binaires orphelins dans ~/bin
  • Multi-arch : amd64, arm64 et armhf — la même installation sur votre bureau et votre Raspberry Pi

🚀 Configurer votre shell

Choisissez un thème dans /usr/share/oh-my-posh/themes/ et initialisez votre shell. Pour bash, ajoutez ceci à ~/.bashrc :

eval "$(oh-my-posh init bash --config /usr/share/oh-my-posh/themes/jandedobbeleer.omp.json)"

Pour zsh, ajoutez la même ligne avec zsh à ~/.zshrc ; pour fish, ajoutez oh-my-posh init fish --config /usr/share/oh-my-posh/themes/jandedobbeleer.omp.json | source à ~/.config/fish/config.fish. La plupart des thèmes s'affichent mieux avec une Nerd Font — ou lancez oh-my-posh font install.
