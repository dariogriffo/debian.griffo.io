Qu'est-ce que Helix ?

Helix est un éditeur de texte modal post-moderne écrit en Rust. Inspiré par Kakoune et Vim, il utilise un modèle d'édition basé sur la sélection et intègre nativement la coloration syntaxique tree-sitter ainsi que la prise en charge du Language Server Protocol — aucun plugin ni configuration n'est nécessaire pour obtenir une expérience proche d'un IDE moderne dans votre terminal. La commande est hx.

🚀 Pourquoi les dernières versions sont importantes : Helix est en développement actif, avec des versions fréquentes qui ajoutent la prise en charge de nouveaux langages, de nouvelles fonctionnalités d'édition et des améliorations de performance. Les dernières versions apportent une meilleure intégration LSP, davantage de thèmes intégrés et des commandes d'édition affinées.

⚡ Fonctionnalités clés de Helix

🌲 Tree-sitter intégré

Coloration syntaxique rapide et précise, et sélections structurelles grâce aux grammaires tree-sitter.

🧠 Serveurs de langage inclus

Autocomplétion, diagnostics et accès direct à la définition fonctionnent immédiatement grâce au LSP.

🎯 Édition centrée sur la sélection

Un modèle de sélections multiples qui rend les modifications complexes intuitives et reproductibles.

⚙️ Zéro configuration

Des réglages par défaut pertinents et des thèmes intégrés — productif dès l'installation.

🎯 Pourquoi choisir Helix ?

  • Tout inclus : Coloration syntaxique et prise en charge LSP sans configuration de plugin
  • Rapide et léger : Écrit en Rust, binaire unique, réactif même sur de gros fichiers
  • Ergonomie modale : Un modèle d'édition raffiné de type sélection puis action
  • Excellents réglages par défaut : Raccourcis clavier faciles à découvrir et palette de commandes interactive

🆕 Points forts

  • LSP intégré : Fonctionnalités d'IDE sans plugins externes
  • Sélections multiples : Modifiez facilement plusieurs endroits à la fois
  • Thèmes : Des dizaines de thèmes de couleurs intégrés
  • Binaire unique : Aucune dépendance d'exécution à gérer

❓ Questions fréquentes

Helix est-il dans les dépôts officiels de Debian ?

Non — Helix n'est pas empaqueté dans les archives officielles de Debian. Ce dépôt est actuellement la seule source apt et propose Helix 25.07.1.

Comment installer le dernier Helix sur Debian ?

Ajoutez le dépôt deb.griffo.io une seule fois (instructions ci-dessus), puis exécutez : sudo apt install helix. Les nouvelles versions arrivent via le sudo apt upgrade habituel.

Les paquets sont-ils signés et comment sont-ils construits ?

Chaque paquet est signé avec la clé GPG du dépôt (EA0F721D231FDD3A0A17B9AC7808B4DD62C41256) et construit à partir des versions upstream dans des dépôts GitHub publics que chacun peut inspecter.

Quelles versions de Debian sont prises en charge ?

Debian 12 Bookworm, Debian 13 Trixie, Forky et Sid.
