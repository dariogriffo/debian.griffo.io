Что такое Atuin?

Atuin заменяет вашу историю команд оболочки удобной для поиска базой данных SQLite, добавляя мощный полнотекстовый поиск, опциональную сквозную зашифрованную синхронизацию между машинами и полезную статистику. Он записывает дополнительный контекст для каждой команды — код завершения, длительность, каталог и многое другое — и работает с Bash, Zsh, Fish и другими оболочками.

🚀 Почему важны последние версии: Atuin активно развивается — с улучшениями в поиске, синхронизации и интерактивном интерфейсе. Использование последней версии даёт вам новейшие функции работы с историей и исправления.

⚡ Ключевые возможности Atuin

🔍 Мощный поиск

Полнотекстовый, нечёткий поиск и поиск по префиксу по всей истории команд оболочки.

🔐 Зашифрованная синхронизация

Опциональная сквозная зашифрованная синхронизация истории между всеми вашими машинами.

📊 Богатый контекст

Записывает код завершения, длительность, каталог и многое другое для каждой команды.

🐚 Поддержка разных оболочек

Работает с Bash, Zsh, Fish и другими популярными оболочками.

🎯 Почему Atuin?

  • Лучшая история: История с возможностью поиска в SQLite, которая никогда не обрезается незаметно
  • Синхронизация между машинами: Сквозная зашифрованная синхронизация с возможностью самостоятельного хостинга
  • Богатый контекст: Знайте, когда, где и как выполнялась каждая команда
  • Статистика: Аналитика по наиболее часто используемым командам

🆕 Основные возможности

  • История в SQLite: Надёжная, доступная для запросов история команд
  • Интерактивный поиск: Нечёткий поиск, привязанный к стрелке вверх
  • Синхронизация: Опциональная зашифрованная синхронизация между несколькими машинами
  • Статистика: Статистика использования команд
