Что такое Zed?

Zed — это редактор кода нового поколения, разработанный для высокопроизводительной совместной работы с людьми и ИИ. Созданный разработчиками Atom и Tree-sitter, Zed написан с нуля на Rust для эффективного использования нескольких ядер CPU и GPU, обеспечивая невероятно быстрый и отзывчивый опыт редактирования.

🚀 Почему важны последние версии: Zed быстро развивается с новыми функциями, такими как нативная поддержка отладки, агентное редактирование, интеграция с Git и улучшения производительности в каждом выпуске. Оставайтесь в курсе для лучшего опыта редактирования.

🔥 Ключевые функции Zed

⚡ Экстремальная скорость

Написан с нуля на Rust для эффективного использования нескольких ядер CPU и GPU. Мгновенный запуск и плавная прокрутка на 120fps.

🤝 Многопользовательское редактирование

Встроенное совместное редактирование в реальном времени. Общайтесь с коллегами, пишите заметки вместе и делитесь экраном и проектом без проблем.

🤖 Интеграция ИИ

Интегрируйте LLM в рабочий процесс с агентным редактированием, встроенным ассистентом и предсказанием редактирования на основе модели Zeta.

🐛 Нативная отладка

Построен на протоколе адаптера отладки (DAP) с нативной поддержкой отладки на нескольких языках программирования.

🔧 Совместимость с Vim

Первоклассное модальное редактирование с помощью клавиш Vim, включая такие функции, как текстовые объекты, метки и все любимые движения.

🌐 Удалённая разработка

Ваша машина запускает только интерфейс Zed, а сама кодовая база работает на удалённом сервере для мощной облачной разработки.
