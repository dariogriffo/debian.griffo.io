Что такое lazygit?

lazygit — это простой терминальный интерфейс для команд git, созданный Jesse Duffield. Он превращает сложные операции Git в интуитивные визуальные взаимодействия, делая контроль версий доступным и эффективным для разработчиков любого уровня. С более чем 63 тысячами звёзд на GitHub, он стал основным Git TUI для разработчиков по всему миру.

😤 Решённые проблемы Git: Устали редактировать TODO-файлы для интерактивного rebasing? Раздражены arcane patch-файлами для staging? Надоели сложные командные рабочие процессы Git? lazygit делает операции Git визуальными, интуитивными и приятными в использовании.
🚀 Почему важны последние версии: lazygit активно разрабатывается с частыми релизами, содержащими новые функции, улучшения производительности и усовершенствования рабочего процесса. Последние версии включают улучшенный интерактивный rebase, лучшее разрешение конфликтов, расширенные пользовательские команды и критические исправления ошибок.

⚡ Ключевые возможности lazygit

📝 Визуальный Staging

Добавляйте отдельные строки или hunks простыми нажатиями клавиш. Больше не нужно редактировать сложные patch-файлы — просто нажмите пробел, чтобы добавить нужное.

🔄 Интерактивный Rebase

Squash, fixup, drop, edit и переупорядочивание коммитов с интуитивными сочетаниями клавиш. Не нужно редактировать TODO-файлы.

🍒 Cherry-picking

Копируйте и вставляйте коммиты между ветками с помощью shift+c и shift+v. Визуальный выбор и применение коммитов.

🔍 Git Bisect

Визуальный интерфейс git bisect для быстрого поиска проблемных коммитов. Помечайте коммиты как хорошие/плохие простыми нажатиями клавиш.

🌳 Worktrees

Создавайте и управляйте Git worktrees для работы над несколькими ветками одновременно без stashing или WIP-коммитов.

⚙️ Пользовательские команды

Расширяемая система пользовательских команд для добавления собственных Git-рабочих процессов и ярлыков прямо в интерфейс.

🚀 Всегда впереди официальных архивов: Debian и Ubuntu замораживают версии пакетов при выпуске релиза — этот репозиторий публикует новые версии обычно в течение нескольких часов после релиза upstream. Простой apt upgrade всегда держит вас на последней версии.

❓ Часто задаваемые вопросы

Есть ли lazygit в официальных репозиториях Debian?

Да, но официальный архив Debian поставляет lazygit 0.57.0, тогда как этот репозиторий предлагает lazygit 0.63.1 — пересобирается через несколько часов после каждого релиза.

Как установить последнюю версию lazygit в Debian?

Добавьте репозиторий deb.griffo.io один раз по инструкции выше, затем выполните: sudo apt install lazygit. Новые версии приходят через обычный sudo apt upgrade.

Подписаны ли пакеты и как они собираются?

Каждый пакет подписан GPG-ключом репозитория (EA0F721D231FDD3A0A17B9AC7808B4DD62C41256) и собирается из релизов upstream в публичных репозиториях GitHub, которые может проверить любой.

Какие версии Debian поддерживаются?

Debian 12 Bookworm, Debian 13 Trixie, Forky и Sid.
