Что такое herdr?

herdr — это мультиплексор агентов, который живёт в вашем терминале — менеджер рабочих пространств для запуска и оркестрации нескольких AI-агентов кодирования бок о бок. Написанный на Rust и поставляемый как единый статический бинарный файл, он позволяет управлять целым парком агентов (Claude Code, Codex, Gemini, Aider и другими) из одного удобного для клавиатуры TUI, с постоянными сессиями, git worktree и удалённым доступом по SSH.

🤖 Создан для AI-агентов кодирования:
  • Запускайте много агентов сразу — каждый в своей панели, вкладке и рабочем пространстве
  • Постоянные сессии, переживающие отключения, как мультиплексор для агентов
  • Осведомлён о git worktree — изолируйте каждого агента на своей ветке
  • Удалённо по SSH — подключайтесь к серверу herdr, работающему где угодно
🚀 Почему важны последние версии: herdr развивается быстро, со стабильным и предварительным каналами и частыми релизами, добавляющими новые интеграции с агентами, функции рабочих пространств и улучшения TUI. Установка из deb.griffo.io держит вас на новейшем релизе без запуска встроенного самообновления.

⚡ Ключевые возможности herdr

🤖 Несколько агентов бок о бок

Запускайте и оркеструйте сразу несколько AI-агентов кодирования, каждый в своей панели и вкладке, и переключайтесь между ними, не покидая терминал.

🗂️ Рабочие пространства и сессии

Группируйте работу в именованные постоянные сессии и рабочие пространства, которые продолжают работать в фоне и мгновенно переподключаются.

🌿 Git worktree

Дайте каждому агенту собственный git worktree и ветку, чтобы параллельная работа никогда не пересекалась, а диффы оставались чистыми и удобными для ревью.

🖥️ Удалённо по SSH

Подключайтесь по SSH к серверу herdr, работающему на любой машине, и управляйте удалёнными сессиями, как если бы они были локальными.

⌨️ TUI с управлением с клавиатуры

Быстрый терминальный интерфейс с опциональной мышью, настраиваемыми сочетаниями клавиш, вкладками и панелями — всё в одном нажатии клавиши.

🔌 Интеграции и расширяемость

Встроенные интеграции с агентами плюс socket API и файл конфигурации, с растущей экосистемой первоклассных плагинов вокруг.

🚀 Всегда впереди официальных архивов: Debian и Ubuntu замораживают версии пакетов при выпуске релиза — этот репозиторий публикует новые версии обычно в течение нескольких часов после релиза upstream. Простой apt upgrade всегда держит вас на последней версии.

❓ Часто задаваемые вопросы

Есть ли herdr в официальных репозиториях Debian?

Нет — herdr не упакован в официальных архивах Debian. Этот репозиторий сейчас единственный источник apt и предлагает herdr 0.7.5.

Как установить последнюю версию herdr в Debian?

Добавьте репозиторий deb.griffo.io один раз по инструкции выше, затем выполните: sudo apt install herdr. Новые версии приходят через обычный sudo apt upgrade.

Подписаны ли пакеты и как они собираются?

Каждый пакет подписан GPG-ключом репозитория (EA0F721D231FDD3A0A17B9AC7808B4DD62C41256) и собирается из релизов upstream в публичных репозиториях GitHub, которые может проверить любой.

Какие версии Debian поддерживаются?

Debian 12 Bookworm, Debian 13 Trixie, Forky и Sid.
