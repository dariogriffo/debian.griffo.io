Что такое Starship?

Starship — это быстрое, настраиваемое кроссшелловое приглашение командной строки, написанное на Rust. Оно работает с Bash, Zsh, Fish, PowerShell и другими оболочками, показывая контекстную информацию — статус git, версии языков, облачный контекст и многое другое — через единое, последовательное и гибко настраиваемое приглашение, которое остаётся быстрым даже в больших репозиториях.

🚀 Почему важны последние версии: Starship активно развивается — добавляются новые модули, интеграции и улучшения производительности. Использование последней версии даёт вам новейшие модули приглашения и параметры конфигурации.

⚡ Ключевые возможности Starship

⚡ Быстрый

Написан на Rust и оптимизирован, чтобы приглашение оставалось отзывчивым везде.

🐚 Поддержка разных оболочек

Единое приглашение для Bash, Zsh, Fish, PowerShell, Nushell и других.

🎨 Полностью настраиваемый

Настройте каждый модуль и символ через единый файл TOML.

🧩 Учитывает контекст

Показывает состояние git, версии языков, облачный контекст и многое другое — только когда это уместно.

🎯 Почему Starship?

  • Единообразие везде: Одно и то же приглашение во всех ваших оболочках и на всех машинах
  • Богатый контекст: Git, языки программирования и окружение — сразу видно
  • Гибкая настройка: Настройте каждую деталь через starship.toml
  • Быстрый: Минимальная задержка даже в больших репозиториях

🆕 Основные возможности

  • Кроссшелловость: Работает в любой популярной оболочке
  • Модули: Десятки встроенных контекстных модулей
  • Конфигурация TOML: Единый переносимый файл конфигурации
  • Пресеты: Готовые оформления, например символы Nerd Font

❓ Часто задаваемые вопросы

Есть ли Starship в официальных репозиториях Debian?

Да, но официальный архив Debian поставляет Starship 1.22.1, тогда как этот репозиторий предлагает Starship 1.26.0 — пересобирается через несколько часов после каждого релиза.

Как установить последнюю версию Starship в Debian?

Добавьте репозиторий deb.griffo.io один раз по инструкции выше, затем выполните: sudo apt install starship. Новые версии приходят через обычный sudo apt upgrade.

Подписаны ли пакеты и как они собираются?

Каждый пакет подписан GPG-ключом репозитория (EA0F721D231FDD3A0A17B9AC7808B4DD62C41256) и собирается из релизов upstream в публичных репозиториях GitHub, которые может проверить любой.

Какие версии Debian поддерживаются?

Debian 12 Bookworm, Debian 13 Trixie, Forky и Sid.
