Что такое Garage?

Garage — это лёгкий, S3-совместимый сервис распределённого объектного хранения. Разработанный для работы на нескольких географически распределённых машинах, он обеспечивает отказоустойчивое, размещаемое на собственной инфраструктуре объектное хранилище с API, совместимым с S3, что делает его надёжной альтернативой с открытым исходным кодом размещённым объектным хранилищам для резервных копий, статических сайтов и данных приложений.

🚀 Почему важны последние версии: Garage активно развивается — с улучшениями репликации, производительности и покрытия S3 API. Использование последней версии даёт вам новейшие функции хранения и исправления надёжности.

⚡ Ключевые возможности Garage

☁️ S3-совместимый API

Прямая совместимость с существующими S3-клиентами, SDK и инструментами.

🌐 Гео-распределение

Реплицируйте данные между несколькими узлами и площадками для отказоустойчивости.

🪶 Лёгкий

Эффективная реализация на Rust, которая хорошо работает на скромном оборудовании.

🔒 Self-hosted

Сохраняйте полное владение своими данными на собственной инфраструктуре.

🎯 Почему Garage?

  • Владейте своим хранилищем: Self-hosted объектное хранилище на собственном оборудовании
  • Экосистема S3: Работает с огромным количеством существующих инструментов S3
  • Отказоустойчивость: Репликация между несколькими узлами переживает отказы узлов
  • Эффективность: Низкое потребление ресурсов, идеально для небольших кластеров

🆕 Основные возможности

  • Объектное хранилище: S3-совместимые бакеты и объекты
  • Репликация: Настраиваемая избыточность между узлами
  • Мульти-дата-центр: Учёт зон при размещении данных
  • Статический хостинг: Раздавайте статические сайты напрямую из бакетов

❓ Часто задаваемые вопросы

Есть ли Garage в официальных репозиториях Debian?

Нет — Garage не упакован в официальных архивах Debian. Этот репозиторий сейчас единственный источник apt и предлагает Garage 2.3.0.

Как установить последнюю версию Garage в Debian?

Добавьте репозиторий deb.griffo.io один раз по инструкции выше, затем выполните: sudo apt install garage. Новые версии приходят через обычный sudo apt upgrade.

Подписаны ли пакеты и как они собираются?

Каждый пакет подписан GPG-ключом репозитория (EA0F721D231FDD3A0A17B9AC7808B4DD62C41256) и собирается из релизов upstream в публичных репозиториях GitHub, которые может проверить любой.

Какие версии Debian поддерживаются?

Debian 12 Bookworm, Debian 13 Trixie, Forky и Sid.
