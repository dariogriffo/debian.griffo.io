Что такое Forgejo Runner?

Forgejo Runner — это CI-раннер для Forgejo Actions. Это лёгкий демон, который подключается к вашему экземпляру Forgejo, забирает задания из очереди рабочих процессов и выполняет их на вашей собственной инфраструктуре — используя Docker, LXC или напрямую на хосте — обеспечивая полностью самостоятельно размещаемый CI/CD-конвейер, совместимый с GitHub Actions.

🚀 Почему важны последние версии: Раннер тесно следует релизам Forgejo, добавляя поддержку новых возможностей рабочих процессов, улучшенную изоляцию и исправления ошибок. Поддержание его в актуальном состоянии обеспечивает совместимость с новейшими возможностями Forgejo Actions.

⚡ Ключевые возможности Forgejo Runner

🔁 Совместимость с Actions

Выполняет рабочие процессы, используя тот же синтаксис и экосистему действий, что и GitHub Actions.

🐳 Гибкое выполнение

Выполняйте задания в контейнерах Docker, LXC или напрямую на хост-машине.

🔐 Самостоятельный хостинг

Держите весь ваш CI/CD полностью на собственной инфраструктуре без зависимости от сторонних сервисов.

⚖️ Масштабируемость

Регистрируйте несколько раннеров с метками для распределения и таргетирования нагрузки.

🎯 Почему Forgejo Runner?

  • Владейте своим CI: Запускайте конвейеры на оборудовании, которое вы контролируете
  • Экосистема Actions: Повторно используйте существующие определения рабочих процессов в стиле GitHub
  • Гибкая изоляция: Docker, LXC или выполнение на хосте — под ваши задачи
  • Легковесность: Единый бинарный файл на Go, который просто развернуть

🆕 Основные особенности

  • Выполнение заданий: Запускает рабочие процессы Forgejo Actions из очереди
  • Метки: Направляйте задания на конкретные раннеры
  • Поддержка контейнеров: Изолированные сборки через Docker
  • Режим демона: Работает непрерывно как сервис
