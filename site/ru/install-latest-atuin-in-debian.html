Что такое Atuin?

Atuin заменяет вашу историю команд оболочки удобной для поиска базой данных SQLite, добавляя мощный полнотекстовый поиск, опциональную сквозную зашифрованную синхронизацию между машинами и полезную статистику. Он записывает дополнительный контекст для каждой команды — код завершения, длительность, каталог и многое другое — и работает с Bash, Zsh, Fish и другими оболочками.

🚀 Почему важны последние версии: Atuin активно развивается — с улучшениями в поиске, синхронизации и интерактивном интерфейсе. Использование последней версии даёт вам новейшие функции работы с историей и исправления.

❓ Часто задаваемые вопросы

Есть ли Atuin в официальных репозиториях Debian?

Да, но официальный архив Debian поставляет Atuin 18.6.1, тогда как этот репозиторий предлагает Atuin 18.17.1 — пересобирается через несколько часов после каждого релиза.

Как установить последнюю версию Atuin в Debian?

Добавьте репозиторий deb.griffo.io один раз по инструкции выше, затем выполните: sudo apt install atuin. Новые версии приходят через обычный sudo apt upgrade.

Подписаны ли пакеты и как они собираются?

Каждый пакет подписан GPG-ключом репозитория (EA0F721D231FDD3A0A17B9AC7808B4DD62C41256) и собирается из релизов upstream в публичных репозиториях GitHub, которые может проверить любой.

Какие версии Debian поддерживаются?

Debian 12 Bookworm, Debian 13 Trixie, Forky и Sid.
