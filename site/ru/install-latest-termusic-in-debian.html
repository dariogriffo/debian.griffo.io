Что такое termusic?

termusic — это полноценный терминальный музыкальный плеер и плеер подкастов, написанный на Rust автором tramhao. Он позволяет свободно слушать музыку и подкасты — как в плане свободы, так и бесплатно. С более чем 1500 звёздами на GitHub, он предоставляет полноценный музыкальный опыт прямо в терминале с обложками альбомов, текстами песен и несколькими аудио-бэкендами.

🆓 Философия свободы:
  • Свобода: Нет зависимости от онлайн-провайдеров или сложных ограничений авторских прав
  • Бесплатно: Скачивайте с YouTube, NetEase, Migu и KuGou без ежемесячной подписки
  • Ваша музыка: Воспроизводите локальную коллекцию без ограничений платформы
🚀 Почему важны последние версии: termusic активно разрабатывается с частыми релизами, содержащими новые функции, улучшения производительности и расширенную поддержку аудиоформатов. Последние версии включают улучшенную поддержку подкастов, улучшенное отображение обложек альбомов и расширенные возможности потоковой передачи.

⚡ Ключевые возможности termusic

🎨 Отображение обложек альбомов

Показывает обложки альбомов прямо в терминале, используя протоколы Kitty, iTerm2, Sixel или ueberzug. Визуальный музыкальный опыт в текстовом режиме.

📻 Поддержка подкастов

Полноценный плеер подкастов с поддержкой RSS-лент, управлением эпизодами и возможностями потоковой передачи.

🔊 Несколько бэкендов

Бэкенды Symphonia (Rust), GStreamer и MPV для максимальной совместимости форматов и вариантов производительности.

📥 Интеграция загрузки

Встроенная поддержка yt-dlp и FFmpeg для загрузки музыки из различных онлайн-источников, включая YouTube.

🏷️ Редактор тегов

Встроенный редактор метаданных для организации музыкальной коллекции. Редактируйте теги, обложки и информацию о файлах прямо в приложении.

🎼 Поддержка текстов

Отображение синхронизированных текстов с настройкой временных меток. Улучшенный музыкальный опыт с просмотром и редактированием текстов.

🎵 Поддерживаемые аудиоформаты

  • Без потерь: FLAC, AIFF, WAV
  • Сжатые: MP3, M4A, Opus, Ogg Vorbis
  • Потоковые: WebM, MKV (зависит от кодека)
  • Метаданные: Полная поддержка тегов для большинства форматов

🆕 Новое в termusic 0.11.0+

  • Enhanced Podcast Support: Улучшенная обработка RSS-лент и управление эпизодами
  • Improved Cover Display: Лучшие протоколы обложек альбомов и совместимость с терминалами
  • Performance Optimizations: Более быстрая обработка аудио и снижение использования памяти
  • Better Tag Editor: Расширенные возможности редактирования метаданных
  • Streaming Improvements: Улучшенная интеграция онлайн-источников и функции загрузки
  • UI Enhancements: Улучшенный интерфейс и пользовательский опыт

🎯 Примеры базового использования

Запустить termusic:

# Start termusic termusic # Start with specific music directory termusic --music-dir ~/Music # Start server mode for remote control termusic-server

Основные элементы управления:

# Playback Space - Play/Pause n - Next track p - Previous track r - Repeat mode s - Shuffle mode # Navigation j/k or ↑/↓ - Move up/down Enter - Select/Play Tab - Switch panels q - Quit # Volume +/- - Volume up/down m - Mute

🚀 Почему выбирают debian.griffo.io?

📊 Сравнение репозиториев:
  • Официальный Debian: Доступен в некоторых версиях, но часто устарел
  • Cargo Install: Требует Rust toolchain и сложные зависимости
  • debian.griffo.io: Последняя версия (0.11.0+) со всеми зависимостями
  • ✅ Последние функции: Получите новейшую поддержку аудиоформатов и возможности плеера
  • ✅ Без компиляции Rust: Предварительно собранные бинарные файлы со всеми бэкендами и зависимостями
  • ✅ Автоматические обновления: Пакеты обновляются в течение нескольких часов после релизов
  • ✅ Полный пакет: Включает все аудио-бэкенды и протоколы обложек
  • ✅ Множество дистрибутивов: Работает на Bookworm, Trixie и Sid
  • ✅ Простое обслуживание: Стандартные команды apt для обновлений
