Что такое lowfi?

lowfi — это маленькое приложение на Rust с единственной целью: воспроизводить lofi-музыку. Созданное talwat, оно делает это максимально просто — никаких альбомов, никакой рекламы, никаких отвлечений, только чистая lofi-музыка для программирования и концентрации. Оно создано в духе философии "suckless", предоставляя именно то, что нужно, и ничего лишнего.

🎯 Философия дизайна: "Я очень не люблю современные музыкальные платформы, и я хотел небольшое приложение в духе 'suckless', которое просто воспроизводит случайный lofi без видео." — talwat, создатель lowfi
🚀 Почему важны последние версии: lowfi активно разрабатывается с частыми релизами, содержащими новые источники треков, улучшения производительности и расширенные функции. Последние версии включают поддержку пользовательских списков треков, интеграцию MPRIS и лучшую сетевую надёжность для непрерывных сессий программирования.

⚡ Ключевые возможности lowfi

🎵 Чистая простота

Приложение с единственной целью — просто воспроизводит lofi-музыку. Никаких сложных интерфейсов, никаких отвлечений, идеально для поддержания состояния потока при программировании.

🌐 Сетевая надёжность

Буферизует 5 целых песен одновременно вместо потоковой передачи фрагментов. Создан для нестабильных сетей и непрерывного прослушивания.

🎛️ Поддержка медиаклавиш

Работает с медиаклавишами и MPRIS (playerctl). Управляйте воспроизведением, не отрываясь от редактора.

📻 Пользовательские списки треков

Поддержка пользовательских списков треков, включая Chillhop, локальные файлы и плейлисты от сообщества.

⚡ Лёгкость

Маленький бинарный файл с минимальным использованием ресурсов. Не замедлит вашу среду разработки.

🎨 Настраиваемый интерфейс

Минималистичный режим, отображение без рамок, поддержка альтернативного экрана и настраиваемая ширина плеера.

🎯 Примеры базового использования

Начать прослушивание:

# Just start lowfi - that's it! lowfi # Start in minimalist mode (no bottom bar) lowfi --minimalist # Start paused lowfi --paused # Use alternate terminal screen lowfi --alternate

Пользовательские списки треков:

# Use Chillhop instead of Lofi Girl lowfi --track-list chillhop # Use custom track list file lowfi --track-list ~/my-lofi-tracks.txt # Use community track lists lowfi --track-list minipop

Расширенные параметры:

# Borderless UI lowfi --borderless # Custom player width lowfi --width 5 # Debug mode (for troubleshooting) lowfi --debug # Custom buffer size lowfi --buffer-size 10

🚀 Почему выбирают deb.griffo.io?

📊 Сравнение установки:
  • Официальный Debian: Недоступен в официальных репозиториях
  • Cargo Install: Требует Rust toolchain и компиляцию
  • Ручной бинарный файл: Ручные обновления без управления пакетами
  • deb.griffo.io: Последняя версия (2.0.7+) с автоматическими обновлениями
  • ✅ Последние функции: Получите новейшие источники треков и улучшения плеера
  • ✅ Без компиляции Rust: Предварительно собранные бинарные файлы, без toolchain
  • ✅ Автоматические обновления: Пакеты обновляются в течение нескольких часов после релизов
  • ✅ Правильная настройка звука: Корректно упакован с зависимостями ALSA и аудио
  • ✅ Множество дистрибутивов: Работает на Bookworm, Trixie и Sid
  • ✅ Простое обслуживание: Стандартные команды apt для обновлений
