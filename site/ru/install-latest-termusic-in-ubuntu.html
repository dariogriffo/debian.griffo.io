Что такое termusic?

termusic — это многофункциональный терминальный плеер музыки и подкастов, написанный на Rust разработчиком tramhao. Он позволяет слушать музыку и подкасты свободно — как в плане свободы, так и бесплатно. С более чем 1500 звёздами на GitHub он обеспечивает полноценное музыкальное взаимодействие прямо в терминале с обложками альбомов, текстами песен и несколькими аудиобэкендами.

🆓 Философия свободы:
  • Свобода: Независимость от онлайн-сервисов и сложных ограничений авторских прав
  • Бесплатность: Загрузка с YouTube, NetEase, Migu и KuGou без ежемесячных подписок
  • Ваша музыка: Воспроизведение локальной коллекции без ограничений платформ
🚀 Почему важны последние версии: termusic активно развивается с частыми выпусками, содержащими новые функции, улучшения производительности и расширенную поддержку аудиоформатов. Последние версии включают улучшенную поддержку подкастов, улучшенное отображение обложек альбомов и расширенные возможности потоковой передачи.

⚡ Ключевые возможности termusic

🎨 Отображение обложек альбомов

Показывает обложки альбомов прямо в терминале с использованием протоколов Kitty, iTerm2, Sixel или ueberzug. Визуальное музыкальное взаимодействие в текстовом режиме.

📻 Поддержка подкастов

Полноценный подкаст-плеер с поддержкой RSS-лент, управлением эпизодами и возможностями потоковой передачи. Слушайте любимые шоу в терминале.

🔊 Несколько бэкендов

Бэкенды Symphonia (Rust), GStreamer и MPV для максимальной совместимости форматов и вариантов производительности.

📥 Интеграция загрузки

Встроенная поддержка yt-dlp и FFmpeg для загрузки музыки из различных онлайн-источников, включая YouTube.

🏷️ Редактор тегов

Встроенный редактор метаданных для организации музыкальной коллекции. Редактирование тегов, обложек и информации о файлах прямо в плеере.

🎼 Поддержка текстов песен

Отображение синхронизированных текстов с настройкой временных меток. Улучшенное музыкальное взаимодействие с просмотром и редактированием текстов.

🎵 Поддерживаемые аудиоформаты

  • Без потерь: FLAC, AIFF, WAV
  • Сжатые: MP3, M4A, Opus, Ogg Vorbis
  • Потоковые: WebM, MKV (в зависимости от кодека)
  • Метаданные: Полная поддержка тегов для большинства форматов
🚀 Всегда впереди официальных архивов: Debian и Ubuntu замораживают версии пакетов при выпуске релиза — этот репозиторий публикует новые версии обычно в течение нескольких часов после релиза upstream. Простой apt upgrade всегда держит вас на последней версии.

🎯 Примеры базового использования

Запустить termusic:

# Start termusic termusic # Start with specific music directory termusic --music-dir ~/Music # Start server mode for remote control termusic-server

Основные клавиши управления:

# Воспроизведение Space - Play/Pause n - Next track p - Previous track r - Repeat mode s - Shuffle mode # Навигация j/k or ↑/↓ - Move up/down Enter - Select/Play Tab - Switch panels q - Quit # Громкость +/- - Volume up/down m - Mute
