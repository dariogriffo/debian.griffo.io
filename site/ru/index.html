⚡ Быстрая установка

Добавьте этот репозиторий в вашу систему Debian и начните устанавливать последние версии ваших любимых инструментов разработки:

Шаг 1: Добавить репозиторий

sudo install -d -m 0755 /etc/apt/keyrings curl -fsSL https://deb.griffo.io/EA0F721D231FDD3A0A17B9AC7808B4DD62C41256.asc | sudo gpg --dearmor --yes -o /etc/apt/keyrings/deb.griffo.io.gpg echo "deb [signed-by=/etc/apt/keyrings/deb.griffo.io.gpg] https://deb.griffo.io/apt $(lsb_release -sc 2>/dev/null) main" | sudo tee /etc/apt/sources.list.d/deb.griffo.io.list > /dev/null sudo apt update

🚀 Почему стоит выбрать deb.griffo.io?

Этот неофициальный репозиторий специализируется на предоставлении самых актуальных версий популярных инструментов разработки в виде правильно упакованных пакетов Debian. Пока официальные репозитории Debian нередко отстают от выпусков upstream, мы гарантируем вам:

  • ✅ Последние стабильные версии в течение нескольких часов после оригинального выпуска
  • ✅ Ночные сборки для передовых функций (Zig, ZLS)
  • ✅ Правильная упаковка Debian с корректными зависимостями
  • ✅ Поддержка нескольких дистрибутивов (Bookworm, Trixie, Forky, Sid)
  • ✅ Автоматические обновления, гарантирующие непрерывную доступность

Область применения: Этот репозиторий сосредоточен исключительно на неофициальной упаковке Debian. По вопросам, связанным с самими инструментами, пожалуйста, обращайтесь к соответствующим исходным проектам.

Обновления: Пакеты автоматически обновляются при выходе новых версий в исходных проектах, гарантируя вам постоянный доступ к последним функциям и исправлениям безопасности.

Сообщество: Это проект, управляемый сообществом, призванный устранить разрыв между быстро развивающимися инструментами разработки и стабильным циклом упаковки Debian.
