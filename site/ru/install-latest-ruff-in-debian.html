Что такое Ruff?

Ruff — это чрезвычайно быстрый линтер и форматер кода Python, написанный на Rust компанией Astral — командой, создавшей uv. Благодаря более чем 800 встроенным правилам и нативным переработкам популярных плагинов один бинарный файл ruff заменяет Flake8, isort, Black, pyupgrade, autoflake и десятки плагинов, работая при этом в 10–100 раз быстрее. Он стал линтером по умолчанию в современной экосистеме Python и используется такими проектами, как FastAPI, Pandas и Hugging Face.

😤 Боли инструментов Python: Управляете flake8 с десятком плагинов, isort воюет с Black, линтинг в CI занимает минуты, а линтеры приходится ставить в каждое виртуальное окружение? Ruff — это один бинарный файл без зависимостей, который делает всё это за миллисекунды, — а установка через apt делает его доступным для всей системы, для каждого проекта и venv, без pip.
🚀 Всегда впереди официальных архивов: Debian и Ubuntu замораживают версии пакетов при выпуске релиза — этот репозиторий публикует новые версии обычно в течение нескольких часов после релиза upstream. Простой apt upgrade всегда держит вас на последней версии.

⚡ Ключевые возможности Ruff

🏎️ В 10–100 раз быстрее

Линтит большие кодовые базы за миллисекунды там, где старым инструментам нужны минуты — достаточно быстро, чтобы запускать при каждом сохранении и в каждом CI-пайплайне.

🧰 Один инструмент, 800+ правил

Заново реализует Flake8, isort, pyupgrade, pydocstyle, eradicate и другие — один конфиг в pyproject.toml заменяет целый стек плагинов.

🖤 Готовая замена форматера

ruff format — форматер, совместимый с Black: тот же стиль, на порядки быстрее, и не нужно устанавливать отдельный инструмент.

🔧 Автоисправления

Сотни правил имеют автоматические исправления: неиспользуемые импорты, обновлённый синтаксис, отсортированные импорты — ruff check --fix, и готово.

🐍 Учёт версий Python

Поддерживает любой Python начиная с 3.7 с поведением правил в зависимости от версии, иерархической конфигурацией, удобной для монорепозиториев, и переопределениями на уровне отдельных файлов.

🔌 Интеграции с редакторами

Встроенный языковой сервер (ruff server) обеспечивает первоклассные интеграции с VS Code, Zed, Neovim и Helix из того же бинарного файла.

🏆 Зачем устанавливать через apt?

  • Для всей системы, без pip: один бинарный файл для каждого проекта, venv, CI-задачи и пользователя — не нужно синхронизировать установки в каждом проекте
  • Идеально для образов Docker/CI: добавьте репозиторий в базовые образы — и каждый пайплайн автоматически получает актуальный ruff
  • Без тулчейна: ни Rust, ни cargo, ни pip — apt-репозиторий с GPG-подписью и готовыми бинарными файлами для 8 архитектур
  • Автоматические обновления: ruff выпускается часто; apt upgrade не отстаёт, и ваши правила никогда не устаревают
  • В паре с uv: пакетный менеджер Astral тоже доступен здесь — полный современный тулчейн Python из одного репозитория

❓ Часто задаваемые вопросы

Есть ли Ruff в официальных репозиториях Debian?

Да, но официальный архив Debian поставляет Ruff 0.0.291, тогда как этот репозиторий предлагает Ruff 0.15.21 — пересобирается через несколько часов после каждого релиза.

Как установить последнюю версию Ruff в Debian?

Добавьте репозиторий deb.griffo.io один раз по инструкции выше, затем выполните: sudo apt install ruff. Новые версии приходят через обычный sudo apt upgrade.

Подписаны ли пакеты и как они собираются?

Каждый пакет подписан GPG-ключом репозитория (EA0F721D231FDD3A0A17B9AC7808B4DD62C41256) и собирается из релизов upstream в публичных репозиториях GitHub, которые может проверить любой.

Какие версии Debian поддерживаются?

Debian 12 Bookworm, Debian 13 Trixie, Forky и Sid.
