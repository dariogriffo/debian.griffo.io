Что такое Zed?

Zed — это редактор кода нового поколения, разработанный для высокопроизводительного взаимодействия с людьми и ИИ. Созданный разработчиками Atom и Tree-sitter, Zed написан с нуля на Rust для эффективного использования нескольких ядер CPU и GPU, обеспечивая невероятно быстрый и отзывчивый опыт редактирования.

🚀 Почему важны последние версии: Zed быстро развивается с новыми функциями, такими как встроенная поддержка отладки, агентное редактирование, интеграция с Git и улучшения производительности в каждом релизе. Оставайтесь в актуальном состоянии для лучшего опыта редактирования.

🔥 Основные возможности Zed

⚡ Чрезвычайно быстрый

Написан с нуля на Rust для эффективного использования нескольких ядер CPU и GPU. Мгновенный запуск и плавная прокрутка со скоростью 120 кадров в секунду.

🤝 Многопользовательское редактирование

Встроенное сотрудничество в реальном времени. Общайтесь с коллегами, пишите заметки вместе и легко делитесь своим экраном и проектом.

🤖 Интеграция ИИ

Интегрируйте LLM в свой рабочий процесс с агентным редактированием, встроенным ассистентом и предсказанием редактирования на основе модели Zeta.

🐛 Встроенная отладка

Основана на протоколе адаптера отладки (DAP) с встроенной поддержкой отладки на нескольких языках программирования.

🔧 Совместимость с Vim

Модальное редактирование первого класса через сочетания клавиш Vim, включая текстовые объекты, метки и все ваши любимые движения.

🌐 Удалённая разработка

Ваша машина выполняет только интерфейс Zed, тогда как реальная кодовая база работает на удалённом сервере для мощной облачной разработки.

❓ Часто задаваемые вопросы

Есть ли Zed в официальных репозиториях Debian?

Нет — Zed не упакован в официальных архивах Debian. Этот репозиторий сейчас единственный источник apt и предлагает Zed 1.11.3.

Как установить последнюю версию Zed в Debian?

Добавьте репозиторий deb.griffo.io один раз по инструкции выше, затем выполните: sudo apt install zed. Новые версии приходят через обычный sudo apt upgrade.

Подписаны ли пакеты и как они собираются?

Каждый пакет подписан GPG-ключом репозитория (EA0F721D231FDD3A0A17B9AC7808B4DD62C41256) и собирается из релизов upstream в публичных репозиториях GitHub, которые может проверить любой.

Какие версии Debian поддерживаются?

Debian 12 Bookworm, Debian 13 Trixie, Forky и Sid.
